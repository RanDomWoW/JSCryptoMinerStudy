	<h1>HoloViews<a class="headerlink" href="#holoviews" title="Permalink to this headline">¶</a></h1>

<div style='display: flex'>
<div style='width: 70%; float: left'>
<p><strong>Stop plotting your data - annotate your data and let it visualize itself.</strong></p>

<p>HoloViews is an <a class="reference external" href="https://github.com/ioam/holoviews/blob/master/LICENSE.txt">open-source</a> Python library designed to make data analysis and visualization seamless and simple.  With HoloViews, you can usually express what you want to do in very few lines of code, letting you focus on what you are trying to explore and convey, not on the process of plotting.</p>
<p>For examples, check out the thumbnails below and the other items in the <a class="reference external" href="gallery/index.html">Gallery</a> of demos and apps and the <a class="reference external" href="reference/index.html">Reference Gallery</a> that shows every HoloViews component.  Be sure to look at the code, not just the pictures, to appreciate how easy it is to create such plots yourself!</p>
<p>The <a class="reference external" href="getting_started/index.html">Getting-Started</a> guide explains the basic concepts and how to start using HoloViews, and is the recommended way to understand how everything works.</p>
<p>The <a class="reference external" href="user_guide/index.html">User Guide</a> goes more deeply into key concepts from HoloViews, when you are ready for further study.</p>
<p>The <a class="reference external" href="Reference_Manual/index.html">API</a> is the definitive guide to each HoloViews object, but the same information is available more conveniently via the <cite>hv.help()</cite> command and tab completion in the Jupyter notebook.</p>
<p>If you have any <a class="reference external" href="https://github.com/ioam/holoviews/issues">issues</a> or wish to <a class="reference external" href="https://help.github.com/articles/about-pull-requests">contribute code</a>, you can visit our <a class="reference external" href="https://github.com/ioam/holoviews">GitHub site</a> or chat with the developers on <a class="reference external" href="https://gitter.im/ioam/holoviews">gitter</a>.</p>

<div class="section" id="installation">
<h1>Installation<a class="headerlink" href="#installation" title="Permalink to this headline">¶</a></h1>
<p><a class="reference external" href="https://pypi.python.org/pypi/holoviews"><img alt="PyPI" src="https://img.shields.io/pypi/v/holoviews.svg" /></a>
<a class="reference external" href="https://anaconda.org/ioam/holoviews"><img alt="Conda" src="https://anaconda.org/ioam/holoviews/badges/installer/conda.svg" /></a>  
  <a class="reference external" href="https://github.com/ioam/holoviews/blob/master/LICENSE.txt"><img alt="License" src="https://img.shields.io/pypi/l/holoviews.svg" /></a> <a class="reference external" href="https://coveralls.io/r/ioam/holoviews"><img alt="Coveralls" src="https://img.shields.io/coveralls/ioam/holoviews.svg" /></a></p>
<p>HoloViews works with <a class="reference external" href="https://travis-ci.org/ioam/holoviews">Python 2.7 and Python 3</a> on Linux, Windows, or Mac, and provides optional extensions for working with the <a class="reference external" href="http://jupyter.org">Jupyter/IPython Notebook</a>.</p>
<p>The recommended way to install HoloViews is using the <a class="reference external" href="http://conda.pydata.org/docs/">conda</a> command provided by <a class="reference external" href="http://docs.continuum.io/anaconda/install">Anaconda</a> or <a class="reference external" href="http://conda.pydata.org/miniconda.html">Miniconda</a>:</p>
<div class="highlight-python"><div class="highlight"><pre><span></span>conda install -c ioam holoviews bokeh
</pre></div>
</div>
<p>This command will install the typical packages most useful with HoloViews, though HoloViews itself
directly depends only on <a class="reference external" href="http://numpy.org">Numpy</a> and <a class="reference external" href="http://ioam.github.com/param/">Param</a>.
Additional installation and configuration options are described in the
<a class="reference external" href="user_guide/Installing_and_Configuring.html">user guide</a>.</p>
<div class="section" id="usage">
<h2>Usage<a class="headerlink" href="#usage" title="Permalink to this headline">¶</a></h2>
<p>Once you&#8217;ve installed HoloViews, you can get a copy of all the examples shown on this website:</p>
<div class="highlight-python"><div class="highlight"><pre><span></span>holoviews --install-examples
cd holoviews-examples
</pre></div>
</div>
<p>And then you can launch Jupyter Notebook to explore them:</p>
<div class="highlight-python"><div class="highlight"><pre><span></span>jupyter notebook --NotebookApp.iopub_data_rate_limit=100000000
</pre></div>
</div>
<p>(Increasing the rate limit in this way is <a class="reference external" href="user_guide/Installing_and_Configuring.html">required for the current 5.0 Jupyter version</a>, but should not be needed in later Jupyter releases.)</p>
