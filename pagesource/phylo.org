					        <h2>
	<strong>The CIPRES Science Gateway V. 3.3</strong></h2>
<p>
	The CIPRES Science Gateway <a href="http://www.phylo.org/index.php/portal/v33">V. 3.3</a> is a public resource for inference of large phylogenetic trees. It is designed to provide all researchers with access to NSF <a href="http://www.xsede.org">XSEDE</a>&#39;s large computational resources through a simple browser interface. <strong>You can now also access these same capabilities programatically with the <a href="http://www.phylo.org/index.php/news/detail/announcing-cipres-restful-services-a-new-way-to-use-cipres">CIPRES REST API</a></strong>.</p>
<p>
	<strong>High Performance Parallel Codes for Large Tree Inference and Sequence Alignment on XSEDE:</strong><br />
	<a href="http://sco.h-its.org/exelixis/software.html">RAxML</a>; <a href="http://mrbayes.sourceforge.net/">MrBayes</a>; <a href="http://beast.bio.ed.ac.uk">BEAST</a>; <a href="http://www.beast2.org">BEAST2</a>; <a href="https://www.nescent.org/wg_garli/Main_Page">GARLI</a>; <a href="http://mafft.cbrc.jp/alignment/software/">MAFFT</a>;&nbsp; <a href="http://phylo.bio.ku.edu/content/tracy-heath-dppdiv">DPPDIV</a>; <a href="http://meta.microbesonline.org/fasttree/">FastTree,</a> <a href="http://code.google.com/p/jmodeltest2/">jModelTest2</a>, <a href="https://people.sc.fsu.edu/~dswofford/paup_test/">PAUP</a>, <a href="http://journals.plos.org/plosone/article?id=10.1371/journal.pone.0070651">ParallelStructure</a>, <a href="http://www.robertlanfear.com/partitionfinder/">PartitionFinder2</a>,&nbsp;<a href="http://www.iqtree.org/">IQ Tree</a>, and <a href="http://popgen.sc.fsu.edu/Migrate/Migrate-n.html">Migrate-N</a>. <i>If you need access to <a href="http://megasun.bch.umontreal.ca/People/lartillot/www/downloadmpi.html">PhyloBayes</a>, please inquire.</i></p>
<p>
	<strong>Serial Codes for Tree Inference:</strong><br />
	<a href="http://paup.csit.fsu.edu/">PAUP*</a> (Inference by Parsimony); <a href="http://research.amnh.org/scicomp/scripts/download.php">Poy</a>&nbsp;(Alignment and Inference);</p>
<p>
	<strong>Serial Codes for&nbsp;Sequence Alignment:</strong><br />
	<a href="http://en.wikipedia.org/wiki/Clustal">ClustalW</a>; <a href="http://contra.stanford.edu/contralign/">Contralign</a>; <a href="http://www.drive5.com/muscle/">MUSCLE</a>; <a href="http://probcons.stanford.edu/">PROBCONS</a>; <a href="http://probalign.njit.edu/probalign/login">PROBALIGN</a></p>
