        <div class="content">
            <h1 class="nmr">About</h1>
        <p>NMR spectroscopy is certainly the analytical methodology that provides the most information about a molecule.
            Teaching and interpreting spectra may however be challenging. On this webpage we have compile many tools that allows to:
            <ul>
                <li>
                    <a href="../new_predictor/">Predict 1D <sup>1</sup>H NMR spectra</a>
                </li>
                <li>
                    <a href="../13c/">Predict 1D <sup>13</sup>C NMR spectra</a>
                </li>
                <li>
                    <a href="../cosy/">Predict COSY spectra</a>
                </li>
                <li>
                    <a href="../hmbc/">Predict HSQC / HMBC spectra</a>
                </li>
                <li>
                    <a href="../simulator/">Simulated second order effect in <sup>1</sup>H NMR spectra</a>
                </li>
                <li>
                    <a href="../resurector/">Recreate (resurect) 1D NMR spectra from experimental in-line assignment</a>
                </li>
            </ul>
	<p>
		<sup>1</sup>H NMR prediction was possible thanks to the tool of the FCT-Universidade NOVA de Lisboa developped by Yuri Binev and Joao Aires-de-Sousa. Y. Binev, M.M. Marques, J. Aires-de-Sousa, <a href="http://dx.doi.org/10.1021/ci700172n">Prediction of 1H NMR coupling constants with associative neural networks trained for chemical shifts</a>  J. Chem. Inf. Model. <b>2007</b>, <i>47</i>/(6), 2089-2097.
	</p>
            <p>This website does not contain any database of NMR spectra but allows to predict easily <sup>13C</sup> as well as <sup>1</sup>H spectra.</p>
