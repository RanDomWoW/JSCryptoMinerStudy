    <h1>Welcome to the Doctrine Project<a class="headerlink" href="#welcome-to-the-doctrine-project" title="Permalink to this headline"></a></h1>

    <p>
        The Doctrine Project is the home to several PHP libraries primarily
        focused on database storage and object mapping. The core projects are a
        <a href="/projects/orm.html">Object Relational Mapper (ORM)</a> and the
        <a href="/projects/dbal.html">Database Abstraction Layer (DBAL)</a> it is
        built upon. Doctrine has greatly benefited from concepts of the
        <a href="http://hibernate.org">Hibernate ORM</a> and has adapted them to fit
        the PHP language.
    </p>

    <h4>Why use Doctrine?</h4>

    <ul>
        <li>Around since 2006 with very stable, high-quality codebase.</li>
        <li>Extremely flexible and powerful object-mapping and query features.</li>
        <li>Support for both high-level and low-level database programming for all your use-cases.</li>
        <li>Large Community and integrations with many different frameworks (Symfony,
            Zend Framework, CodeIgniter, Flow, Lithium and more)</li>
    </ul>
