						<h1>What is TianoCore?</h1>

					</div>
					<div class="tcContentMain">

<p>Welcome to TianoCore, the community supporting an open source implementation of the Unified Extensible Firmware Interface (<a href="https://github.com/tianocore/tianocore.github.io/wiki/UEFI" target="_blank">UEFI</a>). <a href="https://github.com/tianocore/tianocore.github.io/wiki/EDK-II" target="_blank">EDK II</a> is a modern, feature-rich, cross-platform firmware development environment for the UEFI and UEFI Platform Initialization (<a href="https://github.com/tianocore/tianocore.github.io/wiki/PI" target="_blank">PI</a>) specifications. We hope that you’ll review our <a href="https://github.com/tianocore/tianocore.github.io/wiki" target="_blank">wiki</a> documentation, use TianoCore for platform firmware, <a href="https://github.com/tianocore/tianocore.github.io/wiki/Reporting-Issues" target="_blank">report any issues</a> you find, and contribute to the community.</p>

<h2 id="projects-and-downloads">Projects and Downloads</h2>
<p>If you want to compile firmware or utilities, we recommend the <a href="/getting-started.html">Getting Started</a> page. This provides an overview of how to download <a href="https://github.com/tianocore/tianocore.github.io/wiki/Getting-Started-with-EDK-II" target="_blank">EDK II from github</a>, and <a href="https://github.com/tianocore/tianocore.github.io/wiki/Reporting-Issues" target="_blank">reporting issues in Bugzilla</a>.</p>

<h2 id="background">Background</h2>
<p>In June of 2004, Intel announced that it would release the “Foundation Code” of its Extensible Firmware Interface (EFI), a successor to the 16-bit x86 “legacy” PC BIOS, under an open source license. This Foundation Code, developed by Intel as part of a project code named Tiano, was Intel’s “preferred implementation” of EFI. This evolved into EDK, EDK II, and other open source projects under the TianoCore community.</p>

<p>The EFI Specifications were contributed to the United EFI Forum as part of the original UEFI Specifications, which has been adopted by over 200 companies and shipped on millions of compute devices. The UEFI Forum does not endorse any particular implementation, but TianoCore is designed to implement the UEFI and UEFI PI specifications.</p>
