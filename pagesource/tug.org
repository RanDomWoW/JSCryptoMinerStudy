<p><b>The TeX Users Group (TUG)</b> was founded in 1980 to provide an
organization for people who are interested in typography and font
design, and/or are users of the TeX typesetting system invented by <a
href="http://www-cs-faculty.stanford.edu/~knuth/">Donald&nbsp;Knuth</a>.

<p><b>TUG is a not-for-profit organization by, for, and of its
members</b>, also representing the interests of TeX users worldwide.
If you use any TeX-related programs (TeX, <a
href="http://www.latex-project.org/">LaTeX</a>, <a
href="http://www.contextgarden.net/">ConTeXt</a>, <a
href="http://www.math.utah.edu/~beebe/fonts/metafont.html">Metafont</a>,
<a href="/metapost.html">MetaPost</a>, <a
href="http://www.gnu.org/software/texinfo/">Texinfo</a>, <a
href="/interest.html">et al.</a>), please consider <a
href="/join.html">joining TUG</a> (or <a href="/usergroups.html">another
TeX user group</a>).
<a href="/join.html">Memberships</a> and <a
href="https://www.tug.org/donate.html">donations</a> are <a
href="/tax-exempt/">tax-deductible</a> in the US.

<p><b><a href="/aims_ben.html">TUG membership benefits</a></b> include
our journal <a href="/TUGboat"><i>TUGboat</i></a> (available both in
print and online), as well as the TeX software collection (<a
href="/texlive/">TeX Live</a>, <a href="/protext/">proTeXt</a>, <a
href="/mactex/">MacTeX</a>, <a
href="/texlive/texcollection.html">etc.</a>). TUG also runs an <a
href="meetings.html">annual TeX conference</a>, among other activities
(see links at left).

<p id="new"><b>If you're new to TeX</b> and are looking for a system to
install, or have questions to ask, <a href="/begin.html">click&nbsp;here
to get started</a>.

<p id="help"><b>If you're looking for help</b>, you can get community
support via a <a href="http://lists.tug.org/texhax">public mailing
list</a> (texhax), <a href="http://latex-community.org/forum/">forum</a>
(latex-community.org), <a href="http://tex.stackexchange.com/"
rel="nofollow">q&amp;a site</a> (tex.stackexchange.com), or <a
href="/interest.html#doc">in other ways</a>.
