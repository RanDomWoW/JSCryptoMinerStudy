        <h1 class="teal-link-context"><a href="/2016/09/20/cangjie6-1/">蒼頡檢字法字根典型字選</a></h1>
    


            </div>

            <time class="teal-link-context" datetime="2016-09-20T09:56:18.000Z"><a href="/2016/09/20/cangjie6-1/">2016-09-20</a></time>

            
    <div class="tags-row">
        
            <a href="/tags/蒼頡/" class="chip teal">蒼頡</a>
        
            <a href="/tags/倉頡/" class="chip teal">倉頡</a>
        
            <a href="/tags/檢字法/" class="chip teal">檢字法</a>
        
            <a href="/tags/輸入法/" class="chip teal">輸入法</a>
        
    </div>



            
                <div class="entry teal-link-context">
                    <blockquote>
<p><strong>博主按：</strong></p>
<ul>
<li><a href="https://zh.wikipedia.org/wiki/%E8%92%BC%E9%A0%A1%E6%AA%A2%E5%AD%97%E6%B3%95">蒼頡檢字法</a>，坊間亦稱「六代倉頡」，但其與通常所謂「三代倉頡」、<br>「五代倉頡」差異較大，目的非僅供漢字<strong>輸入</strong>，而在完整解決漢字<strong>檢索</strong>及應用問題。</li>
<li>因<a href="https://zh.wikipedia.org/wiki/%E8%92%BC%E9%A0%A1%E6%AA%A2%E5%AD%97%E6%B3%95">蒼頡檢字法</a>至今並無所謂「官方」版本釋出，據稱「原本並不公開，僅供與『朱邦復工作室』有工作關係的人士，及部分文史專家學者使用」。因而本文所依據之字碼方案，係依據 <a href="https://github.com/LEOYoon-Tsaw/Cangjie6">LEOYoon-Tsaw</a> 爲 Rime 輸入法引擎所製蒼頡檢字法輸入方案，此版好處在於製作時參與者眾，使用者又可依自身用字習慣修正使用；但此版與傳說中的「官版」未必一致，當然，「官版」也未必勝過此版。</li>
<li>本文作者使用<a href="https://github.com/LEOYoon-Tsaw/Cangjie6">此版</a>試着逐個輸入了<a href="https://en.wikipedia.org/wiki/Ciyuan">《辭源》</a>所收一萬多個字頭，期間亦曾對異體字、新舊字形輸入之類問題簡單分辨。這一萬多字大概涵蓋先秦古籍所出現的所有字頭，也包括了絕大部分兩漢至清之間（「後起」與「晚起」）產生的常用字。</li>
<li>結果顯示，在輸入方面，<a href="https://github.com/LEOYoon-Tsaw/Cangjie6">此版</a>蒼頡檢字法全部覆蓋上述字頭，且大致能兼顧不同輸入習慣。</li>
<li>如旣有「兼」（的的廿金hhtc：合《說文》字源），也有「兼」（金一止金cmxc：合「舊字形」），也有「兼」（廿止金：合楷化與手寫風格）；旣有「研」（一口一十十），也有「研」（一口一廿）。（各機器與頁面字體支持情形不一，以上三個字形，頁面未必能區別開來）</li>
<li>當然，若使用者不特別修正，此版極個別字碼之定立或可商榷，極個別字碼之定立也未能兼顧其各種字形輸入習慣。如「冒」字碼只有「日月山abu」而無「月月山bbu」，「曼」字碼只有「日田中水awle」而無「月田中水bwle」。</li>
<li>結論：權衡<strong>檢索</strong>效率與<strong>輸入</strong>效率，在三、五、六之間，個人更偏愛「六代」。涵蓋近八萬字頭的碼表，大致滿足古籍、簡體、異體字、新舊字形等各種輸入需求。再也不必爲文獻中出現的某個稀奇古怪的字而煩惱了。</li>
<li>當然，對多數用戶而言，目前許多拼音輸入法也完全可以滿足使用需求，<strong>毫無必要</strong>浪費寶貴的生命年華學習這玩意兒。

<p><strong>博主按：</strong></p>
<ul>
<li>此文純粹爲「安利」學生多寫作業而編造，因某日課後，漢文三年二班某生曰「一個『怎麼就高興了呢』講了兩節課」，有必要記錄整個「說服」以便後續<strong>重複</strong>使用。</li>
<li>一直信奉「師逸而功倍，師勤而功半」的敎條（《禮記·學記》），是以最愛「佈置作業」，每令諸生「叫苦不迭」，亦常引起反彈。今日又起反彈，特在課堂花費諸多口舌以《論語·學而》「安利」之。</li>
<li>雖然「半路出家」研習哲學，對「中國哲學」依然一竅不通，自然不會扯出啥深刻而振聾發聵的理論來；又沒有<strong><em>于丹</em></strong>那般「熬雞湯」的本事，自然也未能引起諸生興趣，「說服」效果不明顯。</li>
<li>是以本文後半部分<strong>全是「瞎扯」</strong>，就是爲敎師偷懶找辯辭——學生不做作業老師怎麼偷懶呢。自然也不會有啥紮實的考證、嚴密的邏輯、有效的結論，<strong>特文前告示之</strong>，語文考試若以本文說法作答風險巨大。
