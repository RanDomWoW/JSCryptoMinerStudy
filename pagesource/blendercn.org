</div></div></div></div><div id="pg-7-2"  class="panel-grid panel-has-style" ><div class="panel-row-style panel-row-style-for-7-2" ><div id="pgc-7-2-0"  class="panel-grid-cell" ><div id="panel-7-2-0-0" class="so-panel widget widget_sow-editor panel-first-child panel-last-child" data-index="5" ><div class="so-widget-sow-editor so-widget-sow-editor-base"><h3 class="widget-title">Cycles 渲染器</h3>
<div class="siteorigin-widget-tinymce textwidget">
	<p>Cycles 是 Blender内置的照片级别的真实渲染器.</p>
<ul>
<li>实时视图预览</li>
<li>CPU 和 GPU 渲染</li>
<li>支持PBR 着色器和 HDR 照明</li>
<li>支持VR 渲染</li>
</ul>
</div>
</div></div></div><div id="pgc-7-2-1"  class="panel-grid-cell" ><div id="panel-7-2-1-0" class="so-panel widget widget_sow-editor panel-first-child panel-last-child" data-index="6" ><div class="so-widget-sow-editor so-widget-sow-editor-base">

<div class="siteorigin-widget-tinymce textwidget">
	<p>Blender齐全的建模工具，让模型创建、变换和编辑变得轻而易举.</p>
<ul>
<li>全面的多边形建模支持</li>
<li>滑动边线, 内插面, 栅格和桥接填充，等等</li>
<li>先进的雕刻工具和笔刷</li>
<li>多级精度和动态细分</li>
<li>使用纹理笔刷和遮罩的三维绘制</li>
<li>插件和用于自定义工具的Python脚本</li>
</ul>
</div>
</div></div></div></div></div><div id="pg-7-4"  class="panel-grid panel-has-style" ><div class="panel-row-style panel-row-style-for-7-4" ><div id="pgc-7-4-0"  class="panel-grid-cell" ><div id="panel-7-4-0-0" class="so-panel widget widget_sow-editor panel-first-child panel-last-child" data-index="9" ><div class="so-widget-sow-editor so-widget-sow-editor-base"><h3 class="widget-title">VFX</h3>
<div class="siteorigin-widget-tinymce textwidget">
	<p>VFX专家说: "这可能是市面上最好的跟踪器了"。Blender包含可用于生产的相机与物体追踪。 可以导入原始镜头，跟踪镜头， 遮蔽局部， 并且在3D场景中实时查看相机运动。完全无需切换其他程序.</p>
<ul>
<li>自动和手动跟踪</li>
<li>强大的相机重建</li>
<li>实时预览跟踪镜头和3D场景</li>
<li>支持平面跟踪和三脚架解算</li>
</ul>
</div>
</div></div></div><div id="pgc-7-4-1"  class="panel-grid-cell" ><div id="panel-7-4-1-0" class="so-panel widget widget_sow-editor panel-first-child panel-last-child" data-index="10" ><div class="so-widget-sow-editor so-widget-sow-editor-base">

<div class="siteorigin-widget-tinymce textwidget">
	<p>得益于高质量的绑定和动画工具，目前Blender被大量用于制作大量短片、广告、TV连续剧和 剧情片.</p>
<ul>
<li>骨骼封套, 骨架和自动蒙皮</li>
<li>B样条曲线插值骨骼</li>
<li>曲线编辑器和动画摄影表</li>
<li>用于快速输入的自定义骨骼形状</li>
<li>音频同步</li>
</ul>
</div>
</div></div></div></div></div><div id="pg-7-6"  class="panel-grid panel-has-style" ><div class="panel-row-style panel-row-style-for-7-6" ><div id="pgc-7-6-0"  class="panel-grid-cell" ><div id="panel-7-6-0-0" class="so-panel widget widget_sow-editor panel-first-child panel-last-child" data-index="13" ><div class="so-widget-sow-editor so-widget-sow-editor-base"><h3 class="widget-title">故事板，3d中绘制2d</h3>
<div class="siteorigin-widget-tinymce textwidget">
	<p>真的！直接在3D视窗绘制的意义重大。 为故事板绘制人员和2D艺术家开创了无与伦比的工作流自由度.</p>
<ul>
<li>在视窗直接结合2D和3D</li>
<li>洋葱皮的全动画支持</li>
<li>笔触和填充的层和调色盘</li>
<li>雕刻笔刷笔触 &amp; 设置父级3D对象</li>
</ul>
</div>

<div class="siteorigin-widget-tinymce textwidget">
	<p>Blender界面可由Python灵活控制。布局、颜色、尺寸甚至字体都可调节。使用社区的数百种插件，或者使用Blender的Python API自制插件.</p>
<ul>

<li>高分屏/Retina屏支持</li>
<li>自行创建工具或插件</li>
<li>在OpenGL视窗进行绘制</li>
<li>Blender渲染API对接</li>
