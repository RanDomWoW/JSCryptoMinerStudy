					<H2>Domain Name Resolution</H2><br /><br />

Domain Name Resolution is the process of figuring out the IP address that corresponds to
a given domain name.
<br /><br />
 For example when you search for more information about domain and
 type into our IP lookup tool 'ipaddresslocation.org'
then our IP tool need to resolve domain name to an IP address. 
<br /><br />
After the process of translating has been finished then you will get an IP address 208.76.83.201.
<br /><br />
Thus IP address that corresponds to domain IPAddressLocation.org is 208.76.83.201
<br />
If you lookup for other a name then you will get other IP address.

<br /><br />
<H2>DNS Lookup - Reverse DNS Lookup</H2>
<br /><br />
DNS, which stands for domain name system, makes possible to associate host names with IP address, 
and conversely. 
<br /><br />
Mostly the DNS is used for the conversion by domainnames in IP addresses 
what is called forward DNS
but also for the conversion by IP addresses in domainnames what is called reverse DNS.
