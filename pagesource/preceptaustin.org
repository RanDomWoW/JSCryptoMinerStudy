    <div class="node__content clearfix">
              
            <div property="schema:text" class="clearfix text-formatted field field--name-body field--type-text-with-summary field--label-hidden field__item"><p><author></author>Welcome to the <a href="http://www.memidex.com/renovate">renovated</a> version of Preceptaustin.org, which is&nbsp;now phone friendly and&nbsp;<a href="https://en.wikipedia.org/wiki/Drupal">database driven</a>&nbsp;allowing&nbsp;us to easily&nbsp;<a href="https://en.wikipedia.org/wiki/Content_management_system">manage content</a> by Bible book and/or topic. Once you get over the shock of the new look, hopefully it will be&nbsp;easier to find all of the resources on a&nbsp;Bible passage or topic of interest (and new resources will be added almost every day). You will also find new search boxes on the right side of most pages, which include&nbsp;various resources such as <a href="https://www.wikipedia.org/">Wikipedia </a>which has a surprisingly large number of topics related to Christianity, albeit it is best used with a&nbsp;healthy "Acts 17:11<a href="/acts_17_commentary#17:11">+ </a>mindset." And don't overlook&nbsp;the useful <a href="http://www.memidex.com/implore">English dictionary search box,</a> which is&nbsp;a&nbsp;superb collection of works&nbsp;which will&nbsp;enable you to hone your prose and preaching with polish and precision. Baker's Evangelical Dictionary of&nbsp;Theology is also an excellent&nbsp;resource to help solve theological <a href="http://www.memidex.com/conundrums">conundrums</a>&nbsp;(<a href="https://www.amazon.com/product-reviews/0801020751/ref=pd_lpo_sbs_dp_ss_1?pf_rd_p=1944687502&amp;pf_rd_s=lpo-top-stripe-1&amp;pf_rd_t=201&amp;pf_rd_i=0801022568&amp;pf_rd_m=ATVPDKIKX0DER&amp;pf_rd_r=HKZFJRYST26ZT491N015">review</a>).&nbsp;</p>

<p class="text-align-center"><strong>HOW TO FIND THE OLD COLLECTIONS&nbsp;<br />
OF&nbsp;SERMONS AND COMMENTARIES</strong></p>

<p>(1) On the old site one could access the <strong>Collections </strong>from a <strong>drop down menu</strong> at the top of the page which allowed one to select the book of interest. This old drop down menu is now on the right side of the page and is designated "<strong>Collections of Commentaries &amp; Sermons</strong>" below which is the button labeled&nbsp;"<strong>Choose Book of the Bible to Study</strong>." Click this tab and you will see a list of hundreds of sermons and commentaries on each book of the Bible. In the old site some of these resources were on a second page, but we are in the process of moving them to one&nbsp;page to facilitate easier access.&nbsp;</p>

<p>(2) Another way to get to the "<strong>Collections of Commentaries &amp; Sermons</strong>" is by&nbsp;clicking&nbsp;the <a href="http://www.preceptaustin.org/bybook"><strong>Commentaries</strong></a>&nbsp;button (Between "Home" and "Topics") and it&nbsp;will take you to a list of all 66 books. Click on a book of the Bible (e.g., <a href="http://www.preceptaustin.org/bybook/50">Philippians</a>) and under "<b>Collections</b>" select "<strong>Philippians</strong>&nbsp;<strong>Commentaries</strong>"&nbsp;to retrieve a compilation of numerous sermons and commentaries on that book of the Bible. The advantage of this method of navigation&nbsp;is that you will also see a list of&nbsp;<strong>Devotionals </strong>and <strong>Miscellaneous </strong>articles that relate to that book of the Bible.</p>

<p>(3) <strong>References or Resources associated with Verse Commentaries</strong> -&nbsp;On the old site, verse commentaries usually had a <a href="http://www.preceptaustin.org/old_reference_table">Reference Table</a> above the commentary notes which provided links to sermons, commentaries or illustrations related to the verse&nbsp;(<a href="http://www.preceptaustin.org/old_reference_table">See picture of old reference table</a>). The old <strong>Reference Table </strong>has been replaced by <b>Resources by Chapter</b>&nbsp;which is&nbsp;found at the top of the page (also on right side of page) of each commentary.&nbsp;For example, select <strong><a href="/philippians_11-8#1:1">Philippians 1:1 Commentary</a></strong>&nbsp; and notice at the top of the page &nbsp;is a link to&nbsp;<a href="http://www.preceptaustin.org/philippians_1_resources"><strong>Philippians 1 Resources</strong></a> which presents multiple related resources identical to those found on the old reference table.&nbsp;There is no listing of <strong>Resources by Chapter</strong> if a&nbsp;<strong>Verse by Verse </strong>commentary has not been written.</p>

<p>(4)&nbsp;The &nbsp;<strong>Devotionals</strong>&nbsp;now update automatically each day.&nbsp;</p>

<p class="text-align-center"><strong>WHY FIX WHAT WAS<br />
NOT BROKEN?</strong></p>

<p><strong>Why did we renovate&nbsp;Preceptaustin.org which did not <a href="http://www.memidex.com/ostensibly">ostensibly</a> seem to be "broken"?</strong> Primarily because&nbsp;the <a href="https://en.wikipedia.org/wiki/HTML">HTML coding</a> had become "<a href="http://www.memidex.com/bloat">bloated</a>" and <a href="http://www.memidex.com/obsolete">obsolete</a> and it seemed increasingly&nbsp;likely&nbsp;that the web hosting company we were using would soon no longer support <strong><a href="https://en.wikipedia.org/wiki/Microsoft_FrontPage">Frontpage</a></strong>,&nbsp;our web authoring system, a "technological dinosaur" that&nbsp;was discontinued by Microsoft over 13 years ago! Indeed, given the atrocious <a href="https://en.wikipedia.org/wiki/HTML">HTML</a> source code Frontpage created, it is clear&nbsp;we have been running on God's amazing <a href="/titus_211-16#gc">grace</a>&nbsp;for quite some time! The final straw was when Microsoft withdrew&nbsp;support for <a href="https://en.wikipedia.org/wiki/Windows_XP">Windows XP</a> which is the system needed to run Frontpage.</p>

<p>Clearly whenever one converts more than 2000 web pages to a&nbsp;new format there may&nbsp;be a few "<a href="https://en.wikipedia.org/wiki/Hiccup">hiccups</a>,"&nbsp;so bear with us as we put the finishing touches on the site. And we <a href="http://www.memidex.com/implore">implore</a> you to please use the&nbsp;<strong>Contact Us</strong>&nbsp;form to let us know if you have problems, and be sure to include the <a href="https://en.wikipedia.org/wiki/Uniform_Resource_Locator">URL</a> of the problem page.&nbsp;We will respond to all comments and/or&nbsp;complaints.&nbsp;</p>

<p>Remember that we have been "<a href="/1corinthians_620_commentary#b">bought with a price</a>" to be "<a href="/2corinthians_520-21_commentary#a">ambassadors</a> for <a href="/1_john_43_commentary#christ">Christ</a>" and our "<a href="/salvation-soteria_greek_word_study">salvation </a>is nearer to us than when we <a href="/1thessalonians_213#believe">believed</a>" so let us "<a href="/hebrews_101-2#cleansed">cleanse</a> ourselves from all <a href="/2corinthians_71#d">defilement</a> of flesh and spirit, <a href="/1_peter_59#accomplished">perfecting</a> <a href="/1thessalonians_311-13#holiness">holiness</a> in the <a href="/ephesians_65-6#f">fear</a> of God" "so that when He <a href="/1_peter_120-21#appeared">appears</a>, we may have <a href="/hebrews_416#c">confidence</a> and not <a href="/1peter_verse_by_verse_410-19#a">shrink away</a> from Him in <a href="/1peter_verse_by_verse_410-19#a">shame</a> at His coming (<a href="http://www.preceptaustin.org/2_peter_115-18#coming">parousia</a>)."&nbsp;<a href="http://www.preceptaustin.org/hallelujah-praise_the_lord">Hallelujah</a>! <a href="http://www.preceptaustin.org/hosanna-save_now">Hosanna</a>!&nbsp;<a href="http://www.preceptaustin.org/maranatha">Maranatha</a>!&nbsp;<a href="http://www.preceptaustin.org/hebrew_definitions#a">Amen</a>! (1 Cor 6:20<a href="/1corinthians_620_commentary">+</a>,&nbsp;2 Cor 5:20<a href="/2corinthians_520-21_commentary#5:20">+</a>,&nbsp;Ro 13:11<a href="/romans_1310-11#13:11">+</a>,&nbsp;2 Cor 7:1<a href="/2corinthians_71#7:1">+</a>,&nbsp;1Jn 2:28<a href="/1john_228-29_commentary#2:28">+</a>)</p>
