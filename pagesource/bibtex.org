<h2 class="themen">Your BibTeX resource</h2>

<p id="abstract">Here you will find everything you need to know about BibTeX</p>
<p>The word ,,<a href="/">BibTeX</a>'' stands for a tool and a file format which are used to describe and process lists of references, mostly in conjunction with LaTeX documents.</p>
<p>Here you can learn about the <a href="Format/">BibTeX File Format</a>, 
<a href="Using/">How to use BibTeX</a> and <a href="Links/">BibTeX Tools</a> which can help you to ease your BibTeX usage.</p>

<p><b>NEW:</b> Be sure to try the <a href="Convert/"><em>Bib2x Online Converter</em></a> which allows you to <b>convert your BibTeX bibliographies</b> into a few target formats. It is meant to serve as a demonstration of <a href="http://www.xandi.eu/bib2x/" class="external">Bib2x</a>, a tool that allows arbitrary conversion of BibTeX bibliographies using templates.</p>
