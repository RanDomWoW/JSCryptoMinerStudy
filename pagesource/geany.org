<h2>Welcome!</h2>
<div><span class='rfloat'> <a class='wikilink' href='https://www.geany.org/Documentation/Screenshots'><img src='https://www.geany.org/images/geany_main_small.png' alt='Geany&#39;s main screen' title='Geany&#39;s main screen' /></a></span></div>
<p>Geany is a text editor using the GTK+ toolkit with basic features of an integrated development environment. It was developed to provide a small and fast IDE, which has only a few dependencies from other packages. It supports many filetypes and has some nice features.
For more details see <a class='wikilink' href='https://www.geany.org/Main/About'>About</a>.
</p>
<p class='vspace'>Latest version is: 1.33
<br clear='all' />
</p>

<div class="blog-title"><p><strong>Geany 1.33 is out!</strong>
</p>
</div>
<div class="blog-timestamp">Posted on Feb. 25, 2018 at 02:32 PM</div>
<div class="blog-permalink"><a class='wikilink' href='https://www.geany.org/Main/20180225'>Permanent link</a></div>

<p>We are happy to announce a new release of Geany!
</p>
<p class='vspace'>For a comprehensive list of changes please see
<a class='wikilink' href='https://www.geany.org/Documentation/ReleaseNotes'>Release Notes</a>.
</p>
<p class='vspace'>Some highlights:
</p>
<div class='vspace'></div><ul><li><span class='wikiword'>GTK3</span> theming improvements and documentation
</li><li>CSS: Update Grid properties
</li><li>Add a note for applying the indent settings in the project preferences
</li><li>Enable popup menu on sidebar and message window notebooks
</li><li>Show status message on attempt to execute empty context action
</li><li>Updated translations: de, el, es, fr, it, lv, pl, pt, tr, ru, zh_CN
</li></ul><p class='vspace'>We want to thank all developers, translators and everyone who
contributed to this release with patches, feedback, bug reports and
so on. Thank you!
</p>
<p class='vspace'>As usual, all downloads can be found on
<a class='wikilink' href='https://www.geany.org/Download/Releases'>Releases</a>.
</p></div>

<div class='vspace'></div><div class="blog-entry2">
<div class="blog-title"><p><strong>Geany-Plugins 1.33 are out!</strong>
</p>
</div>
<div class="blog-timestamp">Posted on Feb. 25, 2018 at 02:31 PM</div>
<div class="blog-permalink"><a class='wikilink' href='https://www.geany.org/Main/20180225'>Permanent link</a></div>

<p>Hello friends and users of Geany,
</p>
<p class='vspace'>We are happy to announce a new version of the combined Geany-Plugins
release -- following the Geany 1.33 release, Geany-Plugins 1.33 is being
released, too.
</p>
<p class='vspace'>Some highlights:
</p>
<div class='vspace'></div><ul><li>Rename plugin <span class='wikiword'>GeanyLaTeX</span> to latex
</li><li>Updated translations: de, es, fr
</li><li>Addons: Show color tip and start Color Chooser with double click
</li><li><span class='wikiword'>GeanyLua</span>: Various imporovments:
<ul><li>Add filetype to set in geany.newfile()
</li><li>Add function geany.reloadconf()
</li></ul></li><li>Pretty Printer:  Allow to pretty print a selection only
</li><li>Spellcheck: Remember 'Check while typing' setting between sessions
</li><li>Spellcheck: Support Enchant 2.0 and later
</li><li>VC: Remember the commit dialog size
</li></ul><p class='vspace'>A more complete list of changes can be found at NEWS at
<a class='urllink' href='https://github.com/geany/geany-plugins/releases/tag/1.33.0' rel='nofollow'>https://github.com/geany/geany-plugins/releases/tag/1.33.0</a>.
</p>
<p class='vspace'>As usual, you can find source tarballs and Windows installer for the new
release on <a class='urllink' href='https://plugins.geany.org' rel='nofollow'>https://plugins.geany.org</a>.
</p>
<p class='vspace'>A huge thanks to everybody contributing to this great release.
</p></div>

<div class='vspace'></div><div class="blog-entry1">
<div class="blog-title"><p><strong>Geany 1.32 is out!</strong>
</p>
</div>
<div class="blog-timestamp">Posted on Nov. 19, 2017 at 12:43 PM</div>
<div class="blog-permalink"><a class='wikilink' href='https://www.geany.org/Main/20171119'>Permanent link</a></div>

<p>We are happy to announce a new release of Geany!
</p>
<p class='vspace'>For a comprehensive list of changes please see <a class='wikilink' href='https://www.geany.org/Documentation/ReleaseNotes'>Release Notes</a>.
</p>
<p class='vspace'>Some highlights:
</p>
<div class='vspace'></div><ul><li>Improve snippet support (visual indicators and more, Thomas Martitz).
</li><li>Improve CLI argument help.
</li><li>Add "Close Documents to the Right" feature.
</li><li>Minor accessibility improvements.
</li><li>Fix crash if plugin manager is opened more than once.
</li><li>Update Python, Assembler and PHP filetypes.
</li><li>Updated translations: ca, de, el, es, fr, it, lt, lv, nl, pt, ru, sk, sv, zh_CN
</li></ul><p class='vspace'>We want to thank all developers, translators and everyone who
contributed to this release with patches, feedback, bug reports and
so on. Thank you!
</p>
<p class='vspace'>As usual, all downloads can be found on <a class='wikilink' href='https://www.geany.org/Download/Releases'>Releases</a>.
</p>
