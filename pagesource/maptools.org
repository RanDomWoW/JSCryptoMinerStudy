                  <h1 align="center">Desktop Tools</h1>

                    <p class="TechNewsHead"><img src="images/bullets/arrowbullet.gif" width="7" height="7"> <strong><a href="http://grass.itc.it/">GRASS</a></strong></p>
                    <p class="TechNewsBody">An open source GIS with raster, topological vector, image processing, and graphics production functionality.</p>

                    <p class="TechNewsHead"><img src="images/bullets/arrowbullet.gif" width="7" height="7"> <strong><a href="http://openev.sourceforge.net/">OpenEV</a></strong></p>
                    <p class="TechNewsBody">A library and reference application for viewing and analysing raster and vector geospatial data.</p>

                    <p class="TechNewsHead"><img src="images/bullets/arrowbullet.gif" width="7" height="7"> <strong><a href="http://qgis.org/">Quantum GIS</a></strong></p>
                    <p class="TechNewsBody">An open source GIS that uses the OGR and GDAL libraries to view several vector and raster formats, and includes support for PostGIS tables and GRASS digitizing.</p>

                    <p class="TechNewsHead"><img src="images/bullets/arrowbullet.gif" width="7" height="7"> <strong><a href="http://udig.refractions.net/">uDig</a></strong></p>
                    <p class="TechNewsBody">An open source viewer/editor that brings together internet, database, and file-based spatial data.</p>

                  <h1 align="center">Utilities and Libraries</h1>

                    <p class="TechNewsHead"><img src="images/bullets/arrowbullet.gif" width="7" height="7"> <strong><a href="http://avce00.maptools.org">AVCE00</a></strong></p>
                    <p class="TechNewsBody">A C library that makes Arc/Info (binary) vector coverages appear as ASCII E00 files.</p>

                    <p class="TechNewsHead"><img src="images/bullets/arrowbullet.gif" width="7" height="7"> <strong><a href="http://dgnlib.maptools.org">DGNLib</a></strong></p>
                    <p class="TechNewsBody">DGNLib is a small C/C++ library for reading and writing DGN files.</p>

                    <p class="TechNewsHead"><img src="images/bullets/arrowbullet.gif" width="7" height="7"> <strong><a href="http://www.gdal.org/">GDAL</a></strong></p>
                    <p class="TechNewsBody">A translator library for raster geospatial data formats.</p>

                    <p class="TechNewsHead"><img src="images/bullets/arrowbullet.gif" width="7" height="7"> <strong><a href="http://trac.osgeo.org/geotiff/">GeoTiff</a></strong></p>
                    <p class="TechNewsBody">A TIFF based interchange format for georeferenced raster imagery.</p>

                    <p class="TechNewsHead"><img src="images/bullets/arrowbullet.gif" width="7" height="7"> <strong><a href="http://mitab.maptools.org">MITAB</a></strong></p>
                    <p class="TechNewsBody">A C++ library to read and write MapInfo .TAB (binary) and .MIF/MID files.</p>

                    <p class="TechNewsHead"><img src="images/bullets/arrowbullet.gif" width="7" height="7"> <strong><a href="http://www.gdal.org/ogr/">OGR</a></strong></p>
                    <p class="TechNewsBody">A C++ open source library (and commandline tools) providing read (and sometimes write) access to a variety of vector file formats.</p>

                    <p class="TechNewsHead"><img src="images/bullets/arrowbullet.gif" width="7" height="7"> <strong><a href="http://trac.osgeo.org/proj/">Proj 4</a></strong></p>
                    <p class="TechNewsBody">A Cartographic Projections library. Proj 4 provides a set of command-line utilities and an API used by several GIS and mapping packages to handle reprojection of coordinates.</p>

                    <p class="TechNewsHead"><img src="images/bullets/arrowbullet.gif" width="7" height="7"> <strong><a href="http://shapelib.maptools.org/">Shapelib</a></strong></p>
                    <p class="TechNewsBody">A C library for reading and writing ESRI Shapefiles.</p>
