        <h1 style="border:0;margin:0;background-image:none">About<br/><em>Bioconductor</em></h1>

        <p style="line-height:1.5em;padding-right:5em">
		Bioconductor provides tools for the analysis and comprehension of
		high-throughput genomic data. Bioconductor uses the R statistical
		programming language, and is open source and open development. It
		has two releases each year,
		<a href="/packages/release/BiocViews.html#___Software">1473 software packages</a>,
		and an active user community. Bioconductor is also
		available as an <a href="/help/bioconductor-cloud-ami/">
		AMI</a> (Amazon Machine Image) and a series of
		<a href="/help/docker/">Docker</a> images.</p>
