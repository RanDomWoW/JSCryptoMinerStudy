<br clear='left' />
<p>Early 2009, I read about KenKen (also known as <b>Calcudoku</b>) in the
New York Times, and decided to write a program that creates similar puzzles, and put them online.
Calcudoku puzzles are similar to Sudoku, but also require some basic math skills to solve.
</p>
<p>The rules are:</p>
<ol>
<li>the 4&times;4 puzzle uses the numbers 1-4, the 6&times;6
the numbers 1-6, and so on</li>
<li>in each row and in each column each digit can
appear only once (just like in Sudoku)</li>
<li>each "cage" (the blocks with the
thick border) shows a result and an operation (addition: +, subtraction: -, multiplication: &times;,
or division: :). The operation applied to the numbers in the cage should
produce the result shown. Note that for subtraction and division the order is not fixed (!)</li>
</ol>
<p>Check out <a href='http://www.youtube.com/watch?v=QC72sAQD7S8' target='_blank'>this video</a> that shows
how to solve two example puzzles.
<p>
You can solve the puzzle on the page using the arrow and number keys, or using the mouse.<br>
Other keys:
<ul>
  <li><b>space bar</b> : clear a cell
  <li><b>backspace</b> : undo a move
  <li><b>=</b> : highlight a number
  <li><b>Shift + number</b> : enter the number as a guess
</ul>
<p>
In the largest puzzles (10x10 Calcudoku puzzle (every Tuesday and Friday) and 12x12 (every Sunday and Thursday)), simply
press 1 then 0, or 1 then 1, or 1 then 2 to enter 10, or 11, or 12. You can also use the a, b, and c keys to enter 10, 11, or 12.
</p> <p>You can also solve the puzzles on the <em>iPhone/iPod Touch/iPad</em>.</p><p>Every puzzle has a single solution.</p><font size=-1>This type of puzzle is known as Calcudoku, Newdoku, Rekendoku, MathDoku, Kashikoku-Naru, KenKen, Kendoku, Sumdoku, Calkuro, K-Doku, Keen, NekNek, CanCan, Square Wisdom, Emono,
Minuplu, LatinCalc, Yukendo, ArithmeGrid, Hitoshii, Inky, SquareLogic, TomTom, and if you know of any other names, let me know <tt>:-)</tt>.
The "no-op" puzzles are sometimes called "Mystery Calcudoku".
Of these names, "KenKen" and "KenDoku" are trademarks of Nextoy LLC. <b>Note</b>
that this is a hobby site, and is not affiliated with Nextoy nor their brands. </font><br><br>

function get_extra_solved_info()
{
  var url9 = "http://www.calcudoku.org/ops/extra_history";

  $.ajax({url: url9,
        type: "POST",
        data: '',
        cache: false,
        timeout: 5000,
        success: function (response) {

//        console.log("extras [" + response + "]");
        
        var data = JSON.parse(response);

        var nr_dates = data.dates.length;
        
        var link_string = "";
        var nr_unsolved = 0;
        for(var i=0; i < nr_dates; i++) {
          var date = data.dates[i];
//          console.log("date " + i + " = " + date + ", solved: " + data.solved[i]);
          
          var link = 'http://www.calcudoku.org/en/' + date + '/extra';
          if (!data.solved[i]) {
            link_string += "<a href='" + link + "'>" + date + "</a>";
            nr_unsolved++;
          }
          $('#unsolved_extras_list').html(link_string);
          $('#unsolved_extras_wrapper').slideDown(500);  // css('display', 'inline');
        }

        $('#unsolved_extras_count').html(nr_unsolved);

//        alert("links [" + link_string + "]");
      }
    });

}  // get_extra_solved_info

function go_to_different_date(year, month, day)
{
  if (month < 10) month = '0' + month;
  if (day < 10) day = '0' + day;
  var new_date = year + '-' + month + '-' + day;  //document.getElementById('datepicker').innerHTML;
  var extra = document.getElementById('extra').value;
  var killer = document.getElementById('killer').value;
  var sudoku = document.getElementById('sudoku').value;
  var user = document.getElementById('user').value;

  var new_link;

  if (user == 1) {
    new_link = 'http://www.calcudoku.org/user/' + new_date + '/en';
  }
  else {
    var tail = '';
    if (extra == 1) tail = 'extra';
    else tail = '4/1';
    
    var head = '';
    if (killer == 1) head = 'killersudoku/';
    else if (sudoku == 1) {
      head = 'sudoku/';
      tail = '9/1';
    }
    
    new_link = 'http://www.calcudoku.org/' + head + 'en/' + new_date + '/' + tail;
  }
  
  window.location = new_link;

}  // go_to_different_date
