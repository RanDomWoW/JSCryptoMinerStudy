<p><strong>How to Pick a Great Domain Name</strong></p>

<p>With all of that in mind, here are a few tips on how to pick a great domain name for your blog.</p>
<ul class="iconlist">
<li>Keep it short and simple</li>
<li><i class="fa fa-plus"></i> Stay away from numbers and dashes</li>
<li><i class="fa fa-plus"></i> Pick a name that is easy to spell and memorable</li>
<li><i class="fa fa-plus"></i> Don’t use any copyright or trademarked names</li>
<li><i class="fa fa-plus"></i> Use keywords that are relevant to your topic</li>
<li><i class="fa fa-plus"></i> Stick with a .com, .net or .org domain extension</li>
<li><i class="fa fa-plus"></i> If it’s available, always try to register YourName.com</li>

<p>How to Install WordPress Manually</p>
<p><strong>Step 1: Download WordPress</strong></p>
<ol>
<li>Download the WordPress package to your local computer from http://wordpress.org/download/.</li>
<li>Unzip the downloaded file to a folder on your local computer.</li>
</ol>
<p><strong>Step 2: Upload WordPress to Your Hosting Account</strong></p>
<p>There are three available options for uploading WordPress to a hosting account.</p>
<ol class="thrv_wrapper">
<li>Upload via FTP &#8211; Please see FTP Getting Started for more information.</li>
<li>Upload via File Manager &#8211; Please see How to Upload Using the File Manager for instructions.</li>
<li>Upload via SSH &#8211; Please see How to Get and Use SSH Access for instructions.</li>
</ol>
<p><strong>Step 3: Create a MYSQL Database and Username</strong></p>
<p>WordPress stores its information in a database. Therefore, a database will need to be created.</p>
<ol>
<li>Log into cPanel.</li>
<li>In the Databases section, click the MYSQL Database Wizard icon.</li>
<li>For Step 1: Create a Database, enter a database name and click Next Step.</li>
<li>For Step 2: Create Database Users, enter the database user name and password and click Create User.</li>
<li>For Step 3: Add User to the Database, click the All Privileges checkbox and click Next Step.</li>
<li>For Step 4: Complete the task, make note of the database name, username and password, as you will need them for Step 4 below.</li>
</ol>
<p><strong>Step 4: Configure wp-config.php</strong></p>
<p>The wp-config-sample.php file has the database information contained within it and tells the WordPress software which database to connect with. This step needs to be completed to make sure the correct database information is connecting with WordPress installation files.</p>
<p>The wp-config-sample.php file can be found in File Manager in the folder where WordPress is installed. To modify the file:</p>
<ol>
<li>Log into cPanel.</li>
<li>In the Files section, click the File Manager icon.</li>
<li>In the pop-up box, select Web Root and check the box for Show Hidden Files (dotfiles), then click Go.<i>If you don&#8217;t see the pop-up box, you may have selected the &#8220;Skip this question, and always open this directory in the future when opening File Manager&#8221; option. To make the pop-up reappear, simply click reset all interface settings at the very bottom of your cPanel.</i></li>
<li>From the left navigation menu in File Manager, click public_html to expand the folder.</li>
<li>Click the folder containing the WordPress installation.</li>
<li>On the right side of the File Manager screen, locate the wp-config-sample.php file.</li>
<li>Right click on the file, select Rename and change the name of the file to wp-config.php. Click the Rename File button.</li>
<li>Right-click on the wp-config.php file and select Code Edit. When the file opens, look for the following information:<i>define(&#8216;DB_NAME&#8217;, &#8216;database_name_here&#8217;);</i>/** MySQL database username */<br />
define(&#8216;DB_USER&#8217;, &#8216;username_here&#8217;);/** MySQL database password */<br />
define(&#8216;DB_PASSWORD&#8217;, &#8216;password_here&#8217;);</li>
</ol>
<ol>
<li>Replace database_name_here with the name of the database you created (above in Step 3: Create MySQL Database and User See Below).</li>
<li>Replace username_here with the username of the database you created.</li>
<li>Replace password_here with the password of the database that you created.When done editing, click the Save Changes button, then close the file.</li>
</ol>
<p><strong>Step 5: Run the Installation</strong></p>
<p>Next, you will want to go to the installation page through a new browser so you can successfully run the installation script. Depending on where you installed the script, you will find it at either of the following URLs:</p>
<ul>
<li>http://yourdomain.com/wp-admin/install.php</li>
<li>http://yourdomain.com/blog/wp-admin/install.phpBe sure to replace yourdomain.com in the example above with your actual domain name.</li>
</ul>
<p>You should now see a welcome page that says &#8220;Welcome to the famous five minute WordPress installation process!&#8221;</p>
<p><strong>Step 6: Run the Installation</strong></p>
<p>On this page, fill in the fields for:</p>
<ul class="thrv_wrapper">
<li>Site Name</li>
<li>Username</li>
<li>Password (needs to be entered twice)</li>
<li>Email address (login information will be sent to this email address)</li>
<li>Select whether or not to have the search engines index the site</li>
</ul>
<p style="font-size: 16px;">Click Install Now, and you should be taken to the final screen, showing the following message:</p>
<p><span style="font-weight: bold; font-style: italic; text-decoration: underline;">Success! WordPress has been installed.</span></p>

<h3><span style="color: #ff6600;">BONUS: Frequently Asked Questions When Blogging</span></h3>
<p>&nbsp;</p>
<p>When it comes to doing anything for the first time, we all have questions. To help with this process, I wanted to provide you with some of the most frequent questions I get asked on a daily basis.</p>
<p><strong>How can I get more visitors to my blog?</strong></p>
<p>Did you know that there are currently over a billion active websites and blogs on the internet today? It&#8217;s true&#8230; and while there are plenty of people already on the internet, that is still a lot of websites and content to compete against. When launching a new website or blog, traffic and visitors to your site might be a bit slow. However, as you continue to come out with new content on your site, rank in the search results and share content with your friends, traffic to your site will continue to grow over time. If you want to implement some actionable methods to start increasing traffic to your site today, get started with these <a href="https://blogging.org/blog/15-free-methods-to-increase-traffic-to-your-site-right-now/" data-wpel-link="internal">15 free traffic generation methods</a>.</p>
<p><strong>When will my site start showing in Google?</strong></p>
<p>Google will automatically find your website or blog once it&#8217;s been around for a few days and has some content published and people visiting your site. A great way to speed up this process is to link to your blog from your social profile or anywhere else that might be relevant. Keep in mind, just because your site has been indexed by Google, that doesn&#8217;t mean it will necessarily start ranking high in the search results or for generic keywords that you think it should. The process of ranking a site in Google is quite complex. I recommend you read through these <a href="https://blogging.org/blog/simple-seo-tips-bloggers/" data-wpel-link="internal">simple SEO tips for beginners</a> to have a better understanding of how this process works.</p>
<p><strong>How can I make money with my blog?</strong></p>
<p>The ultimate goal for many bloggers is to eventually make money with their site and content. Fortunately, this is something I&#8217;ve had a lot of experience with. So much so, that I actually wrote a complete guide on <a href="https://blogging.org/blog/how-to-make-money-blogging/" data-wpel-link="internal">how to make money blogging</a>. Not only does it break down the process of how to choose a niche market or different monetization methods to use, I also show you real examples of how this is being done today. If you want to find financial success with blogging, it&#8217;s all about providing value to your audience. Billions of people are using Google and visiting sites on a daily basis to find solutions to their problems. If your website or blog can help solve those problems, there is a good chance you can start making some money in the process.</p>
<p><strong>Can I get famous from starting a blog?</strong></p>
<p>Often times we will see everyday people become overnight success stories through the use of websites, social media, YouTube and even blogs. The first thing to note is that most of these &#8216;overnight&#8217; success stories aren&#8217;t really overnight and have a lot of blood, sweat, and tears behind them. Take Perez Hilton for example, he got started with a <a href="https://blogging.org/blog/how-to-start-a-celebrity-gossip-blog/" data-wpel-link="internal">celebrity gossip blog</a> and look how rich, famous and successful he is today. In short, you can definitely can get famous or become and industry leader within your market through the use of a blog. Depending on what you are trying to accomplish with your site, a lot of things can happen. For instance, I was featured on Fox News and ABC News as a blogging expert and have been able to teach thousands of people how to get started with a blog of their own. No matter what it is you are trying to accomplish with a blog, if you put in the time and effort to make it great, you can definitely become famous, successful and even wealthy in the process!</p>
