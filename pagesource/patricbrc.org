                            <h2 class="wp-headline section-title section-title-b tools_headline"><span class="wrap">What Can I Do @ PATRIC?</span>
                            </h2>
                            <div>
                                <div class="column"
                                     style="width: 311px;display:inline-block;vertical-align:top;float:left;margin-right:10px;">
                                    <ul class="no-decoration no-underline-links">
                                        <li class="clear">
                                            <figure class="figure-right">
                                                <span class="sr-only">Genome Assembly</span>
                                                <i aria-hidden="true" class="home_wcid home_wcid-assembly"></i>
                                            </figure>
                                            <h3 class="bold"><a href="/app/Assembly">Genome Assembly</a></h3>
                                            <p>The Genome Assembly Service App can be used to perform an automatic
                                                genome assembly using the latest computational tools. Single or multiple
                                                assemblers can be invoked to compare
                                                results.&nbsp;<strong>Note:&nbsp;</strong><em>You must have a PATRIC
                                                    user account and be logged in to use this service.</em>
                                            </p>
                                        </li>
                                        <li class="clear">
                                            <figure class="figure-right">
                                                <span class="sr-only">Protein Family Sorter</span>
                                                <i class="home_wcid home_wcid-protein_family_sorter"></i>
                                            </figure>
                                            <h3 class="bold">
                                                <a href="/app/ProteinFamily">Protein Family Sorter
                                                </a>
                                            </h3>
                                            <p>Compares protein families across closely related or diverse groups of
                                                genomes, visualizes them using interactive heatmaps, and generates
                                                multiple sequence alignments and phylogenetic trees for individual
                                                families. The heatmap view is an interactive visualization tool that
                                                provides an overview of the distribution of proteins across a selected
                                                set of genomes.
                                            </p>
                                        </li>
                                        <li class="clear">
                                            <figure class="figure-right">
                                                <span class="sr-only">Genome Metadata</span>
                                                <i class="home_wcid home_wcid-genome_metadata"></i>
                                            </figure>
                                            <h3 class="bold">
                                                <a href="/view/Taxonomy/2#view_tab=genomes">Genome Metadata
                                                </a>
                                            </h3>
                                            <p>Supports searching for and locating genomes of interest based on various
                                                combinations of 61 different metadata fields. For instance, all genomes
                                                that have been isolated from humans, genomes related by phylogeny, or
                                                genomes related by lifestyle.
                                            </p>
                                        </li>
                                    </ul>
                                </div>
                                <div class="column last">
                                    <ul class="no-decoration no-underline-links">
                                        <li class="clear">
                                            <figure class="figure-right">
                                                <span class="sr-only">Genome Annotation</span>
                                                <i class="home_wcid home_wcid-genome_annotation"></i>
                                            </figure>
                                            <h3 class="bold"><a href="/app/Annotation">Genome Annotation</a></h3>
                                            <p>The Genome Annotation Service is a fully-automated pipeline for
                                                annotating bacterial and archaeal genomes using RAST (Rapid Annotation
                                                using Subsystem Technology) to provide high quality genome annotations.&nbsp;<strong>Note:&nbsp;</strong><em>You
                                                    must have a PATRIC user account and be logged in to use this
                                                    service.</em></p></li>
                                        <li class="clear">
                                            <figure class="figure-right">
                                                <span class="sr-only">Comparative Pathway Tool</span>
                                                <i class="home_wcid home_wcid-comparative_pathways"></i>
                                            </figure>
                                            <h3 class="bold"><a href="/app/ComparativePathway">Comparative Pathway
                                                    Tool</a></h3>
                                            <p>Supports comparison of consistently annotated metabolic pathways across
                                                closely related or diverse groups of genomes and visualizes them using
                                                interactive KEGG maps and heatmaps. The heatmap view is an interactive
                                                visualization&nbsp;tool that provides an overview of the distribution of
                                                genomes across the set of EC numbers within a selected pathway.</p></li>
                                        <li class="clear">
                                            <figure class="figure-right">
                                                <span class="sr-only">Gene Page Transcriptomics</span>
                                                <i class="home_wcid home_wcid-gene_page_transcriptomics"></i>
                                            </figure>
                                            <h3 class="bold"><a
                                                        href="/view/Feature/PATRIC.83332.12.NC_000962.CDS.2949593.2950486.fwd#view_tab=transcriptomics">Gene
                                                    Page Transcriptomics</a></h3>
                                            <p>Summarizes all comparisons in which a gene is differentially expressed
                                                and associated metadata such as Strain, Gene Modification, and
                                                Experimental Condition.&nbsp;Expression value based filters and metadata
                                                graphical summaries enable quick discovery of top experimental
                                                conditions and gene manipulations related to the differential expression
                                                of the gene. <strong>Note:</strong> <i>Links to example Gene Page for </i><i>VBIMycTub87468_2933</i><i>.&nbsp;</i>
                                            </p></li>

                        <h2 class="section-title section-title-b"><span class="wrap">Recent News</span></h2>
                        
                        <div style="text-align:left;margin:0px;margin-top:10px;margin-left:4px;"><span
                                    style="padding:4px;color:#555;font-weight:normal;">2/12/2018</span>
                        </div>
                        <div style="border:0px; border-top:1px solid #ddd;text-align:center;padding-top:6px;margin:4px;margin-bottom:8px;margin-top:0px;">
                            <div style="font-size:1.3em;text-align:left;padding-left: 20px;color:#000;"><a
                                        href="https://docs.patricbrc.org/news/2018/20180213-metagenomic-binning-webinar.html" target="_blank" rel="noopener">PATRIC Webinar – Metagenomic Binning Service, Wednesday February 28, 2018, 3:00pm EST (20:00 GMT)</a></div>
                            <div style="text-align:left;padding-left:20px;"><div class="compound">
<p class="compound-first">PATRIC provides a new Metagenomic Binning Service that accepts either reads or contigs, and attempts to “bin” the data into a set of genomes. It can be used to reconstruct bacterial and archael genomes from environmental samples. The new service will produce reports that estimate the quality of the reconstructed genome for each bin, as well as potentially problematic annotations for each of the derived genomes.</p>
<p class="compound-middle">On February 28th at 3pm EST we will provide a webinar demonstrates this service. Please email <a class="reference external" href="mailto:rwattam&#64;vt.edu">rwattam&#64;vt.edu</a> if you plan to attend so that we will know approximately how many participants to expect. The webinar will be recorded and made available for later access.</p>
<p class="compound-last"><strong>Note: The recorded webinar is now available here:</strong> &lt;<a class="reference external" href="https://docs.patricbrc.org//webinar/metagenomic_binning.html">https://docs.patricbrc.org//webinar/metagenomic_binning.html</a>&gt;</p>
</div></div>
                        </div>
                        
                        <div style="text-align:left;margin:0px;margin-top:10px;margin-left:4px;"><span
                                    style="padding:4px;color:#555;font-weight:normal;">2/5/2018</span>
                        </div>
                        <div style="border:0px; border-top:1px solid #ddd;text-align:center;padding-top:6px;margin:4px;margin-bottom:8px;margin-top:0px;">
                            <div style="font-size:1.3em;text-align:left;padding-left: 20px;color:#000;"><a
                                        href="https://docs.patricbrc.org/news/2018/20180206-proteome-comparison-video-tutorial.html" target="_blank" rel="noopener">PATRIC Video Tutorial – Proteome Comparison</a></div>
                            <div style="text-align:left;padding-left:20px;"><div class="compound">
<p>PATRIC provides a service that allows researchers to compare annotated proteins by bi-directional BLASTP analysis. The Proteome Comparison tool can be used to examine up to compare nine genomes to single reference, which can be a genome that has been annotated in PATRIC or from an independent source, or a group of proteins. A recorded webinar that demonstrates this service is now available on the PATRICBRC YouTube site at &lt;<a class="reference external" href="https://youtu.be/sozy1mJh1EI">https://youtu.be/sozy1mJh1EI</a>&gt;</p>
</div></div>

                                        href="https://docs.patricbrc.org/news/2018/20180131-patric-january-2018-release.html" target="_blank" rel="noopener">PATRIC January 2018 Data and Website Release</a></div>
                            <div style="text-align:left;padding-left:20px;"><div class="compound">
<p>In this release, PATRIC has added over 4,000 new genomes and associated metadata, bringing the total number of genomes and plasmids in PATRIC to over 150,000. This release also features a new Metagenomic Binning Service that allows users to assemble and annotate high-quality microbial genomes directly from metagenomic reads.</p>
