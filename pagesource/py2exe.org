<h1 id="py2exe">py2exe</h1>
<span class="anchor" id="line-6"></span><span class="anchor" id="line-7"></span><p class="line862">py2exe is a <a class="http" href="http://www.python.org/">Python</a> <a class="http" href="http://www.python.org/doc/current/distutils/">Distutils</a> extension which converts Python scripts into executable Windows programs, able to run without requiring a Python installation.<a href="http://www.py2exe.org/cgi-bin/exothermic"><img alt="Spice" class="external_image" src="http://www.py2exe.org/spice.gif" title="Spice" width="0" /></a> <span class="anchor" id="line-8"></span><span class="anchor" id="line-9"></span><p class="line862">Development is hosted at <a class="http" href="http://sourceforge.net/projects/py2exe/">SourceForge</a>. You can find the <a class="https" href="https://lists.sourceforge.net/lists/listinfo/py2exe-users">mailing list</a>, <a class="http" href="http://sourceforge.net/svn/?group_id=15583">svn</a>, and <a class="http" href="http://sourceforge.net/projects/py2exe/files/py2exe/0.6.9/">downloads</a> there. <span class="anchor" id="line-10"></span><span class="anchor" id="line-11"></span><p class="line874">py2exe was originally developed by Thomas Heller who still makes contributions. Jimmy Retzlaff and Mark Hammond have also made contributions. Code contributions are always welcome from the community and many people provide invaluable help on the mailing list and the Wiki. <span class="anchor" id="line-12"></span><span class="anchor" id="line-13"></span><p class="line862">py2exe is used by <a class="http" href="http://www.bittorrent.com">BitTorrent</a>, <a class="http" href="http://spambayes.sourceforge.net/windows.html">SpamBayes</a>, and thousands more - py2exe averages over 5,000 downloads per month. <span class="anchor" id="line-14"></span><span class="anchor" id="line-15"></span><p class="line874">In an effort to limit Wiki spam, this front page is not editable. Feel free to edit other pages with content relevant to py2exe. You will need an account to edit (but not to read) and your account information will be used only for the purposes of administering this Wiki. <span class="anchor" id="line-16"></span><span class="anchor" id="line-17"></span><p class="line862">The old <a class="http" href="http://www.py2exe.org/old">py2exe</a> web site is still available until that information has found its way into this wiki. <span class="anchor" id="line-18"></span><span class="anchor" id="line-19"></span><p class="line867">
