<h1 class="sectionedit1" id="home">Home</h1>
<div class="level1">

<p>
<a href="http://freedomdefined.org/OSHW" class="media" target="_blank" title="http://freedomdefined.org/OSHW"  rel="nofollow"><img src="/lib/exe/fetch.php?tok=984a4a&amp;media=http%3A%2F%2Fwww.openhardwaresummit.org%2Fwp-content%2Fuploads%2F2011%2F08%2Foshw-logo-100-px.png" class="mediaright" alt="" /></a>
</p>

<p>
Pixhawk is an independent, open-hardware project aiming at providing high-end autopilot hardware to the academic, hobby and industrial communities at low costs and high availability. It provides hardware for the Linux Foundation <a href="http://dronecode.org" class="urlextern" target="_blank" title="http://dronecode.org"  rel="nofollow">DroneCode</a> project. It originated from the <a href="http://pixhawk.ethz.ch" class="urlextern" target="_blank" title="http://pixhawk.ethz.ch"  rel="nofollow">PIXHAWK Project</a> of the <a href="http://cvg.ethz.ch" class="urlextern" target="_blank" title="http://cvg.ethz.ch"  rel="nofollow">Computer Vision and Geometry Lab of ETH Zurich (Swiss Federal Institute of Technology)</a> and  <a href="http://www.asl.ethz.ch" class="urlextern" target="_blank" title="http://www.asl.ethz.ch"  rel="nofollow">Autonomous Systems Lab</a> as well from a number of excellent invididuals (<a href="/credits" class="wikilink1" title="credits">Contact and Credits</a>).
</p>

<p>
<p><div class="notetip">
<strong>We are always looking for new developers!</strong> Get in touch via our <a href="http://discuss.px4.io/" class="urlextern" target="_blank" title="http://discuss.px4.io/"  rel="nofollow">Discuss Server</a> (discuss.px4.io).
</div></p>
</p>

</div>

<h2 class="sectionedit2" id="autopilot_software">Autopilot Software</h2>
<div class="level2">

<p>
The Pixhawk autopilot module runs a very efficient real-time operating system (RTOS), which provides a POSIX-style environment (i.e. printf(), pthreads, /dev/ttyS1, open(), write(), poll(), ioctl(), etc). The software can be updated with an USB bootloader. We support multiple flight stacks: <a href="http://px4.io" class="urlextern" target="_blank" title="http://px4.io"  rel="nofollow">PX4</a> and <a href="http://ardupilot.org" class="urlextern" target="_blank" title="http://ardupilot.org"  rel="nofollow">APM</a>.
</p>

</div>

<h2 class="sectionedit3" id="autopilot_hardware">Autopilot Hardware</h2>
<div class="level2">

<p>
<a href="/modules/pixhawk" class="wikilink1" title="modules:pixhawk">PIXHAWK</a> is the industry standard autopilot, designed by the open hardware development team in collaboration with <a href="https://3dr.com" class="urlextern" target="_blank" title="https://3dr.com"  rel="nofollow">3D Robotics</a> and <a href="http://www.ardupilot.org" class="urlextern" target="_blank" title="http://www.ardupilot.org"  rel="nofollow">ArduPilot Group</a>.
</p>

<p>
<a href="/modules/pixhawk" class="media" title="modules:pixhawk"><img src="/_media/modules/pixhawk-logo-view.jpg?w=400&amp;tok=331821" class="medialeft" alt="" width="400" /></a>
<br/>

</p>
<ul>
<li class="level1"><div class="li"> 168 <abbr title="Megahertz">MHz</abbr> Cortex M4F CPU (256 KB RAM, 2 <abbr title="Megabyte">MB</abbr> Flash)</div>
</li>
<li class="level1"><div class="li"> Sensors: 3D ACC / Gyro / MAG / Baro</div>
</li>
<li class="level1"><div class="li"> Integrated backup, override and failsafe processor with mixing</div>
</li>
<li class="level1"><div class="li"> microSD slot, 5 UARTs, CAN, I2C, SPI, ADC, etc</div>
</li>
<li class="level1"><div class="li"> Order from:<br/>
</div>
</li>
</ul>
