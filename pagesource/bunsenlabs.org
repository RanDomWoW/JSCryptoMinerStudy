The project is a community continuation of <a href="https://en.wikipedia.org/wiki/CrunchBang_Linux">CrunchBang Linux</a>.</p>

        <h2 id="defining-features">Defining features</h2>
        <p>The distribution consists of configuration and resource packages installed on top of Debian. There are no changes to the way the Debian base system is administrated.</p>
        <ul>
        <li>Pre-configured <a href="http://openbox.org/wiki/Main_Page">Openbox</a> window manager with <a href="https://gitlab.com/o9000/tint2">tint2</a> panel and <a href="http://conky.sourceforge.net/">conky</a> system monitor</li>
        <li>Assortment of harmonising GTK2/3 themes, wallpapers and <a href="https://github.com/brndnmtthws/conky">conky</a> configurations</li>
        <li>Various configuration and application utilities to maintain this system</li>
        <li>Additional desktop-, multimedia- and hardware-related packages come pre-installed to offer a better “out-of-the-box” experience.</li>
        </ul>
        <p>There is no limit to customization: You can review desktop configurations in our forums’ <a href="https://forums.bunsenlabs.org/viewforum.php?id=8">Artwork &amp; Screenshots</a> section, and we also host <a href="https://forums.bunsenlabs.org/viewforum.php?id=9">several active threads</a> on conky configuration.</p>
        <h2 id="architectures">Architectures</h2>
        <p>We offer combined <a href="/installation.html#live-cddvdusb-key-method">live-CD and installation media</a> for the Debian <strong>i386</strong> (i686) and <strong>amd64</strong> (x86-64) architectures.</p>
        <p>However, you can install our <a href="/installation.html#package-based-install">packages</a> also on ARM platforms. Our repository currently provides <strong><a href="https://wiki.debian.org/ArmHardFloatPort">armhf</a></strong>. If you would like to see <a href="https://www.debian.org/ports/arm/">another Debian ARM port</a> supported, file a request <a href="https://forums.bunsenlabs.org/viewforum.php?id=13">here</a>.</p>
        <h2 id="community-support">Community support</h2>
        <p>In general, we recommend new users to <a href="https://forums.bunsenlabs.org">visit and sign up for our forums</a>: It provides plenty of resources and assistance in dealing with every configuration aspect of the BL distribution, and is the main platform for user-to-user support as well as the place to voice suggestions and report bugs.</p>
        <p>There are a few independent communities, see <a href="https://www.bunsenlabs.org/faq.html#independent-channels">here for a list</a>.</p>
