<h2 id="about-systemrescuecd">About SystemRescueCd</h2>
<p><strong>Description:</strong> SystemRescueCd is a Linux <strong>system rescue disk</strong> available as a
bootable CD-ROM or <a href="/Installing-SystemRescueCd-on-a-USB-stick/">USB stick</a> for
administrating or repairing your system and data after a crash. It aims to
provide an easy way to carry out admin tasks on your computer, such as creating
and editing the hard disk partitions. It comes with a lot of
<a href="/System-tools">Linux system utilities</a> such as GParted, fsarchiver, filesystem
tools and basic tools (editors, midnight commander, network tools). It can be
used for both Linux and
<a href="/manual/Backup_data_from_an_unbootable_windows_computer/">windows</a>
computers, and on desktops as well as servers. This rescue system requires no
installation as it can be booted from a CD/DVD drive or
<a href="/Installing-SystemRescueCd-on-a-USB-stick/">USB stick</a>, but it can be
<a href="/manual/Installing_SystemRescueCd_on_the_disk/">installed on the hard disk</a>
if you wish. The kernel supports all important file systems (ext3/ext4, xfs,
btrfs, reiserfs, jfs, vfat, ntfs), as well as <strong>network filesystems</strong> such as
Samba and NFS.</p>

<h2 id="system-tools-included">System tools included</h2>
<ul>
<li><a href="http://www.gnu.org/software/parted/"><strong>GNU Parted</strong></a>: creates, resizes, moves, copies partitions, and filesystems (and more).</li>
<li><a href="http://gparted.org/"><strong>GParted</strong></a>: GUI implementation using the GNU Parted library.</li>
<li><a href="http://www.fsarchiver.org/"><strong>FSArchiver</strong></a>: flexible archiver that can be used as both system and data recovery software</li>
<li><a href="http://www.partimage.org/"><strong>Partimage</strong></a>: popular opensource disk image software which works at the disk block level</li>
<li><a href="http://www.gnu.org/software/ddrescue/"><strong>ddrescue</strong></a> : Attempts to make a copy of a partition or floppy/Hard Disk/CD/DVD that has hardware errors, optionally filling corresponding bad spots in input with user defined pattern in the copy.</li>
<li><strong>File systems tools</strong> (for Linux and Windows filesystems): format, resize, and debug an existing partition of a hard disk</li>
<li><a href="/manual/Mounting_ntfs_filesystems/"><strong>Ntfs3g</strong></a>: enables read/write access to MS Windows NTFS partitions.</li>
<li><a href="http://www.partimage.org/Partimage-manual_Backup-partition-table"><strong>sfdisk</strong></a> <strong>saves and restores</strong> partition tables.</li>
<li><a href="http://www.cgsecurity.org/wiki/TestDisk"><strong>Test-disk</strong></a> : tool to check and undelete partition, supports reiserfs, ntfs, fat32, ext3/ext4 and many others</li>
<li><a href="/manual/System_boot_floppy_disks/"><strong>Memtest</strong></a>: to test the memory of your computer (first thing to test when you have a crash or unexpected problems)</li>
<li><a href="/manual/Backup_and_transfer_your_data_using_rsync/"><strong>Rsync</strong></a>: very-efficient and reliable program that can be used for remote backups.</li>
<li><strong>Network tools</strong> (Samba, NFS, ping, nslookup, &hellip;): to backup your data across the network</li>
</ul>
<p>Browse the <a href="/System-tools/">short system tools page</a> for more details about the most important software included.</p>

<h2 id="advanced-uses">Advanced uses</h2>
<p>SystemRescueCd is available for blind people. Now, the linux speakup screen
reader is working well, and the speakup keymap is installed. This feature was
tested by Gregory Nowak. Notes specific to using speakup in SystemRescueCD are
available on the <a href="/Speakup-info/">speakup info page</a></p>
<p><strong>It is possible to <a href="/Customization/">make customized versions</a> of the system.</strong>
For example, you can add your own scripts, make an automatic restoration of the
system. It is also possible to <strong><a href="/manual/Burning_a_bootable_DVD_with_SystemRescueCd/">burn a customized DVD</a></strong>, with
SystemRescueCd and 4GB for your data (backup for example). Read the manual for
more details.</p>
<p><strong>You can use SystemRescueCd to <a href="/manual/Backup_data_from_an_unbootable_windows_computer/">backup data from an unbootable Windows computer</a></strong>, if you
want to backup the data stored on a Windows computer that cannot boot any more.</p>
<p><strong>It is very easy to <a href="/Installing-SystemRescueCd-on-a-USB-stick/">install SystemRescueCd on a USB stick</a></strong>. That is very useful in case you
cannot boot from the CD drive. You just have to copy several files to the stick
and run syslinux. The install process can be done from Linux or Windows. Follow
instructions from the <a href="/manual/">manual</a> for more details.</p>
