			<div class="news_entry">
                <h2><a href="https://www.khronos.org/news/press/khronos-group-releases-vulkan-1-1">Khronos Group Releases Vulkan 1.1</a></h2>
                
                <p>The Khronos Group announces the release of the Vulkan 1.1 and SPIR-V 1.3 specifications. Version 1.1 expands Vulkan’s core functionality with developer-requested features, such as subgroup operations, while integrating a wide range of proven extensions from Vulkan 1.0. Khronos will also release full Vulkan 1.1 conformance tests into open source and AMD, Arm, Imagination, Intel Corporation, NVIDIA and Qualcomm have implemented conformant Vulkan 1.1 drivers. <br />
<br><br />
Vulkan now ships natively on almost all GPU-enabled platforms, including Windows 7, 8.X, 10, Android 7.0+ and Linux, plus Khronos recently announced open source tools to enable Vulkan 1.0 applications to be <a href="https://www.khronos.org/news/press/vulkan-applications-enabled-on-apple-platforms">ported to macOS and iOS</a>. Vulkan has widespread support in leading games engines including Unreal, Unity, Source 2 from Valve, id Tech, CroTeam’s Serious Engine, CryEngine, and Xenko. Vulkan is being used in over 30 cutting-edge games on diverse desktop and mobile platforms, including Doom, Quake, Roblox, The Talos Principle, Dota 2, and is the exclusive API used in AAA titles such as Wolfenstein II and Doom VFR.<br />
<br><br />
Read the <a href="https://www.khronos.org/news/press/khronos-group-releases-vulkan-1-1">press release</a> and learn more about the Vulkan 1.1 specification and associated tests and tools on the <a href="https://www.khronos.org/vulkan">Vulkan home page</a>.</p>
                <p class="small-text"><small>Mar 07, 2018 | <a href="https://www.khronos.org/news/press/khronos-group-releases-vulkan-1-1">Read article...</a> | <a href="/news/comments/khronos-group-releases-vulkan-1.1">Permalink</a></small></p>
            </div><div class="news_entry">
                <h2><a href="https://www.khronos.org/news/press/vulkan-applications-enabled-on-apple-platforms">Vulkan Applications Enabled on Apple Platforms</a></h2>
                
                <p>The Khronos Group announces that the Vulkan Working Group&#8217;s Portability Initiative has been working with Khronos members Valve, LunarG, and The Brenwill Workshop to enable Vulkan applications to be ported to Apple platforms. The <a href="https://www.khronos.org/vulkan/portability-initiative">Vulkan Portability resource page</a> links to a collection of free and open source set of tools, SDKs, and runtime libraries to enable Vulkan development on macOS and deployment on macOS and iOS platforms. Valve is extending Dota 2 using the Vulkan tools on macOS to <a href="https://www.youtube.com/watch?v=IwFC1NXsSZM">achieve significantly higher performance</a> than native OpenGL drivers. Vulkan support for Dota 2 on macOS will be released in the coming months as a free update.</p>
                <p class="small-text"><small>Feb 26, 2018 | <a href="https://www.khronos.org/news/press/vulkan-applications-enabled-on-apple-platforms">Read article...</a> | <a href="/news/comments/vulkan-applications-enabled-on-apple-platforms">Permalink</a></small></p>
            </div><div class="news_entry">
                <h2><a href="https://www.cginternals.com/en/blog/2018-01-10-screen-aligned-quads-and-triangles.html">Screen-filling Rasterization using Screen-aligned Quads and Triangles with OpenGL</a></h2>
                
                <p>CG Internals published a blog article covering screen-filling rasterization using graphics hardware and modern OpenGL. The findings are applicable to OpenGL ES, Vulkan, and WebGL as well. For rendering screen-filling geometry we usually have to choose between a screen-aligned quad and a screen-aligned triangle. But - is there a difference? If so, which approach is better than the other? In this article we want to show you the differences between both approaches and offer an alternative. Following the theoretical analysis we introduce a demo program and evaluate screencasts together with multiple performance measures.</p>
                <p class="small-text"><small>Feb 25, 2018 | <a href="https://www.cginternals.com/en/blog/2018-01-10-screen-aligned-quads-and-triangles.html">Read article...</a> | <a href="/news/comments/screen-filling-rasterization-using-screen-aligned-quads-and-triangles-with">Permalink</a></small></p>
            </div><div class="news_entry">
                <h2><a href="https://blogs.igalia.com/itoral/2018/02/01/intel-mesa-driver-for-linux-is-now-opengl-4-6-conformant/">Intel Mesa driver for Linux is now OpenGL 4.6 conformant</a></h2>
                
                <p>Khronos announced the conformance program for OpenGL 4.6 and Intel has successfully passed conformance tests for its GPU models for the Mesa Linux driver. For specifics on the conformant hardware you can check the <a href="https://www.khronos.org/conformance/adopters/conformant-products/opengl">list of conformant OpenGL products</a> at the Khronos website. Learn more about these drivers and the OpenGL conformance tests and why they are important to you.</p>
                <p class="small-text"><small>Feb 01, 2018 | <a href="https://blogs.igalia.com/itoral/2018/02/01/intel-mesa-driver-for-linux-is-now-opengl-4-6-conformant/">Read article...</a> | <a href="/news/comments/intel-mesa-driver-for-linux-is-now-opengl-4.6-conformant">Permalink</a></small></p>
            </div><div class="news_entry">
                <h2><a href="https://www.khronos.org/news/press/khronos-launches-opengl-4.6-adopters-program">Khronos Launches OpenGL 4.6 Adopters Program - Conformance Test Suite in Open Source</a></h2>
                
                <p>The Khronos Group announces the launch of the OpenGL 4.6 Adopters Program to enable implementations to become officially conformant to the latest generation OpenGL specification. The significantly enhanced OpenGL 4.6 Conformance Test Suite used in the Adopters Program has been released in open source on GitHub to enable industry participation in testing and ongoing conformance test suite improvements. General information on Khronos Adopters Programs can be found <a href="https://www.khronos.org/conformance/adopters/">here</a>.</p>
                <p class="small-text"><small>Jan 31, 2018 | <a href="https://www.khronos.org/news/press/khronos-launches-opengl-4.6-adopters-program">Read article...</a> | <a href="/news/comments/khronos-launches-opengl-4.6-adopters-program-conformance-test-suite-in-open">Permalink</a></small></p>
