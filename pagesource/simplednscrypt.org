<h1 id="simple-dnscrypt">Simple DNSCrypt</h1>
<p>Simple DNSCrypt is a simple management tool to configure <a href="https://github.com/jedisct1/dnscrypt-proxy">dnscrypt-proxy</a> on windows based systems.</p>

<h2 id="status">Status</h2>

<p>New version based on dnscrypt-proxy 2.0.6</p>

<h2 id="getting-started">Getting Started</h2>

<h3 id="prerequisites">Prerequisites</h3>

<p>At least one system with Windows 7 SP1 and the installation of. NET Framework 4.6.1 is currently required.</p>

<p>You also will need: Microsoft Visual C++ Redistributable for Visual Studio 2017 <a href="https://aka.ms/vs/15/release/VC_redist.x64.exe">x64</a> or <a href="https://aka.ms/vs/15/release/VC_redist.x86.exe">x86</a></p>

<h3 id="installing">Installing</h3>

<p>To install Simple DNSCrypt use the latest (stable) MSI packages: <a href="https://github.com/bitbeans/SimpleDnsCrypt/releases/download/0.5.3/SimpleDNSCrypt.msi">x86</a> or <a href="https://github.com/bitbeans/SimpleDnsCrypt/releases/download/0.5.3/SimpleDNSCrypt64.msi">x64</a>.</p>

<h3 id="file-signing">File Signing</h3>

<p>The MSI package and the SimpleDnsCrypt.exe are signed via a <em>COMODO RSA Code Signing CA</em>. 
The files are signed under the name: <em>EAM Experience Area MÃ¼nsingen GmbH</em></p>

<p>You also can verify the MSI packages with <a href="https://jedisct1.github.io/minisign/">minisign</a>.
The minisign signatures <a href="https://github.com/bitbeans/SimpleDnsCrypt/releases/download/0.5.3/SimpleDNSCrypt.msi.minisig">x86</a> and <a href="https://github.com/bitbeans/SimpleDnsCrypt/releases/download/0.5.3/SimpleDNSCrypt64.msi.minisig">x64</a> can be verified with the following command:</p>

<h3 id="deinstallation">Deinstallation</h3>

<p>To uninstall Simple DNSCrypt and dnscrypt-proxy, just go to the Windows Control Panel (Programs and Features) and search for Simple DNSCrypt.</p>

<h3 id="updates">Updates</h3>

<p>Simple DNSCrypt will automatically search for the latest version at startup.</p>
