<p>
<a href="https://www.omnimaga.org/news/gameboy-emulators-for-the-casio-prizm-and-ti-84-ce/"><strong>Gameboy Emulators for the Casio Prizm and TI-84+ CE</strong></a> by <a href="https://www.omnimaga.org/profile/pimathbrainiac/">pimathbrainiac</a><br />
<span class="smalltext">May 14, 2017, 10:34:14 pm</span>
</p>
</div>Hello! It&#039;s been a while since this news editor has written anything, and it&#039;s time to change that! Without further ado, let&#039;s jump right in to it!<br /><br />First up, for the Casio Prizm, we&#039;ll be discussing <strong>Prizoop</strong>, a Gameboy/Gameboy color emulator. Developed by community newcomer <a href="https://www.cemetech.net/forum/profile.php?mode=viewprofile&amp;u=18715" class="bbc_link" target="_blank">tswilliamson</a>, it claims to run approximately 95% of Gameboy games at 60 frames per second, with &quot;safe&quot; software overclocking enabled. Sound support, ROM compression, and auto frameskip are new additions in the latest update. The project is open source and the code is available on GitHub for anyone who wants to contribute or just view the code. Planned features include savestates, as well as performance and file footprint improvements.<br /><br />Second, the long awaited CE port of the ever popular TI-83+ series gameboy emulator, TI-Boy, has been announced, although no downloads or source code are available at this time. <strong>TI-Boy CE</strong>, by long time community member <a href="https://www.omnimaga.org/profile/calc84maniac/" class="bbc_link" target="_blank">calc84maniac</a>, appears to be coming along nicely, with a 60 fps <a href="https://www.youtube.com/watch?v=9drK96SJU_Q" class="bbc_link" target="_blank">video</a> showing off a working version of the program. Planned features include automatic colorization of supported Gameboy games, real time clock emulation, and a configuation saving feature.<br /><br />As always on Omnimaga, we do not recommend nor condone the usage of pirated ROMs. That said, if anyone has any Gameboy or Gameboy Color homebrew that they have written or would like to share, feel free to link it below, so long as the site hosting the homebrew does not host any sort of pirated material.<br /><br />If you wrote some homebrew for the Gameboy or Gameboy Color, feel free to message me a screenshot or gif of it running on one of the featured emulators, or any emulator running on a calculator. You might be featured on a follow up article!<br /><br />Thanks for reading!
