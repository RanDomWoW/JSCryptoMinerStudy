        <p>The Joda project provides quality low-level libraries for the Java platform.</p>

<h4 id="projects">Projects</h4>

<p>These are the projects:</p>

<ul>
  <li><a href="http://www.joda.org/joda-time/">Joda-Time</a> - Basic types for Date and Time</li>
  <li><a href="http://www.joda.org/joda-money/">Joda-Money</a> - Basic types for Money</li>
  <li><a href="http://www.joda.org/joda-beans/">Joda-Beans</a> - Next generation JavaBeans</li>
  <li><a href="http://www.joda.org/joda-convert/">Joda-Convert</a> - String to Object conversion</li>
  <li><a href="http://www.joda.org/joda-collect/">Joda-Collect</a> - Additional collection data structures</li>
</ul>

<p>Each project aims to be focussed on a specific area and to have minimal dependencies.</p>

<h4 id="pronunciation">Pronunciation</h4>
<p>The ‘J’ in ‘Joda’ is pronounced the same as the ‘J’ in <a href="http://dictionary.reference.com/browse/Java">Java</a>.
 The project is not capable of wielding a <a href="http://en.wikipedia.org/wiki/Yoda">light saber</a> and is thus pronounced differently.</p>
