          <h2 id="is-rubyinstaller-for-you">Is RubyInstaller for you?</h2>

<p>RubyInstaller is the easiest and most used Ruby environment on Windows.
And <a href="https://www.ruby-lang.org">Ruby</a> is a <a href="http://www.bestprogramminglanguagefor.me/why-learn-ruby">great language</a> for beginners as well as professionals.
It’s suitable for small scripts as well as large applications.
RubyInstaller combines the possibilities of native Windows programs with the rich UNIX toolset of <a href="http://www.msys2.org">MSYS2</a> and the <a href="https://github.com/Alexpux/MINGW-packages">large repository of MINGW libraries</a>.
It is a great foundation to use Ruby for development and production
