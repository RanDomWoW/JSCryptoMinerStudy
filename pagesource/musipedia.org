<div style="background-image:url(typo3temp/GB/9279f05879.gif); background-repeat:no-repeat; background-position:left top;"><h1>The Open Music Encyclopedia</h1></div><div class="dottedline"></div><p>Welcome to Musipedia! Inspired by, but not affiliated with <a href="http://wikipedia.org" target="_blank">Wikipedia</a>, we are building a searchable, editable, and expandable collection of tunes, melodies, and musical themes.</p>
<p>Every entry can be edited by anybody. An entry can contain a bit of sheet music, a MIDI file, textual information about the work and the composer, and last but not least the Parsons Code, a rough description of the melodic contour.</p><div style="background-image:url(typo3temp/GB/f9f88df0e1.gif); background-repeat:no-repeat; background-position:left top;"><h1>Name that Tune</h1></div><div class="dottedline"></div><div class="csc-textpic csc-textpic-intext-right"><div class="csc-textpic-imagewrap csc-textpic-single-image"><img src="uploads/pics/mh.gif" width="136" height="90" border="0" alt="" /></div><div class="csc-textpic-text"><p>Musipedia uses the <a href="http://www.melodyhound.com" target="_blank">&quot;Melodyhound&quot; melody search engine.</a> You can <a href="melody_search.html" target="_self">find and identify a tune</a> even if  the melody is all you know. You can <a href="flash_piano.html" target="_self">play it on a piano keyboard</a>, <a href="microphone.html" target="_self">whistle it to the computer</a>, simply <a href="query_by_tapping.html" target="_self">tap the rhythm on the computer keyboard</a> or <a href="melodic_contour.html" target="_self">use</a> the Parsons code.</p></div></div><div class="csc-textpic-clear"><!-- --></div><div style="background-image:url(typo3temp/GB/5c2d5e5b4a.gif); background-repeat:no-repeat; background-position:left top;"><h1>Music Search</h1></div><div class="dottedline"></div><p>You can base your search on melody (i. e., pitch and rhythm), melodic contour, or just rhythm.</p><div style="background-image:url(typo3temp/GB/f52bbae1a3.gif); background-repeat:no-repeat; background-position:left top;"><h2>Melody Search</h2></div><p>Identify a melody by entering it</p><ul><li>with a <a href="flash_piano.html">Flash-based piano</a>,</li><li>a <a href="js_piano.html">JavaScript-based piano</a>,</li><li>by dragging <a href="draw_notes.html">the mouse</a>,</li><li>or whistle it to the computer using <a href="microphone.html">a microphone</a>.</li></ul><div style="background-image:url(typo3temp/GB/f52e97e89b.gif); background-repeat:no-repeat; background-position:left top;"><h2>Contour-based Search</h2></div><p>If you are unsure about the exact intervals between notes, try the <a href="melodic_contour.html" target="_self">Melodic Contour Search</a>.
