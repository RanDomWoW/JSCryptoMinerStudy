				<H1>
					<SPAN CLASS="highlight">Question2Answer</SPAN> is a free and open source platform for Q&amp;A sites.
				</H1>
			
				<DIV CLASS="twocolumn">
					<H2>What is a Question and Answer site?</H2>
					
					<P>A Q&amp;A site helps your online community to share knowledge. People with questions get the answers they need. The community is enriched by commenting, voting, notifications, points and rankings.</P>

					<H2>Why offer Q&amp;A on my site?</H2>

					<P>Your members will enjoy the interaction that Q&amp;A enables, and will visit your site more regularly. In addition, many web searches are questions, so Q&amp;A content will attract search engine traffic.</P>
					
					<H2>What is Question2Answer?</H2>
					
					<P>Question2Answer (Q2A) is a popular open source Q&amp;A platform for PHP/MySQL, currently running on <A HREF="sites.php">22,012 sites in 40 languages</A></A>.</P>
					
					<H2>How do I get Q2A?</H2>
					
					<P><A HREF="question2answer-latest.zip"><B>Download Question2Answer 1.8.0</B></A>, then read <A HREF="install.php">how to install</A>. Version 1.8.0 was <A HREF="versions.php">released</A> on February 8, 2018. Also <A HREF="https://github.com/q2a/question2answer">on GitHub</A>.
					<!--<DIV STYLE="padding-top:10px; font-size:85%;"><A HREF="versions.php">Q2A 1.8.0 beta 2</A> was released on December 23, 2017. </DIV>-->
					</P>

					<H2>What does Q2A need?</H2>
					
					<UL>
						<LI><A HREF="http://www.php.net/">PHP</A> 5.2 or later, with the <A HREF="http://en.wikipedia.org/wiki/MySQLi">MySQLi</A> extension.</LI>
						<LI><A HREF="http://www.mysql.com/">MySQL</A> 4.1 or later, MySQL 5.x for best performance.</LI>					
					</UL>
					
					<H2>What's the license?</H2>
					
					<P><SPAN CLASS="highlight">Question2Answer is open source</SPAN>, <A HREF="license.php">licensed under GPL v2+</A>.</P>

					<H2>How can I help?</H2>
					
					<P>Contribute <A HREF="https://github.com/q2a/question2answer">via Github</A> or create a <A HREF="themes.php">theme</A>, <A HREF="plugins-tutorial.php">plugin</A> or <A HREF="translate.php">translation</A>.</P>

					<!--<H2>Isn't Q&amp;A just like an online forum?</H2>

					<H2>
						Easy to set up
					</H2>
					
					<UL>
						<LI>Q2A takes 5 minutes (or less!) to <A HREF="install.php">install</A>.</LI>
						<LI>Easy styling with <A HREF="themes.php">CSS themes</A>.</LI>
						<LI>Supports <A HREF="translate.php">translation</A> into any language.</LI>
						<LI>Custom sidebar, widgets, pages and links.</LI>
						<LI>SEO features such as <A HREF="http://sitemaps.org/">XML Sitemap</A> and neat URLs.</LI>
					</UL>
					
					<H2>
						Fast and secure
					</H2>
					
					<UL>
						<LI>PHP/MySQL <A HREF="optimize.php">scalable</A> to millions of users and posts.</LI>
						<LI><A HREF="secure.php">Safe from</A> XSS, CSRF and SQL injection attacks.</LI>
						<LI>Beat spam with captchas, moderation and/or flagging.</LI>
					</UL>
					
					<H2>
						Core Q&amp;A features
					</H2>
					
					<UL>
						<LI>Fast integrated search engine.</LI>
						<LI>Categories (up to 4 levels deep) and/or tagging.</LI>
						<LI>Voting, comments, follow-on and closed questions.</LI>
						<LI>Points-based reputation management.</LI>
						<LI>RSS, email notifications and personal news feeds.</LI>
					</UL>
					
					<H2>
						Built-in user account management...
					</H2>
						
					<UL>
						<LI>Create experts, editors, moderators and admins.</LI>
						<LI>User avatars (or <A HREF="http://www.gravatar.com/">Gravatar</A>) and custom fields.</LI>
						<LI>Private messages and public wall posts.</LI>
						<LI>Log in via Facebook or others (using <A HREF="addons.php#plugins">plugins</A>).</LI>
					</UL>

					<H2>
						...or integrate with existing sites
					</H2>
					
					<UL>
						<LI>Out-of-the-box <A HREF="wordpress.php">WordPress 3+ integration</A>.</LI>
						<LI>Custom <A HREF="single-sign-on.php">single sign-on</A> support for other sites.</LI>
						<LI>Deep integration and customization via <A HREF="plugins.php">plugins</A>.</LI>
					</UL>
