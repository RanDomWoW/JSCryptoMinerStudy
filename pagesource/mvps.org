      <h1 class="Subhead"><a name="dev"></a>Application Development</h1>
      <dl>
        <dt>Alex Angelopoulos' <a href="/scripting/">Windows Scripting Site</a></dt>
        <dd>Resources for Windows scripters, focusing primarily (but not exclusively) on WSH.  Topics include extension of scripting with COM, tools for scripters, performance analysis, numerical scripting, scripting and programming interoperability, scripting philosophy, and the ever-popular Rube Goldberg memorial scripting page commemorating bizarre scripting solutions.</dd>

        <dt>Randy Birch's <a href="http://vbnet.mvps.org/">VB<i>net</i></a></dt>
        <dd>VB<em>net</em> is Randy's Visual Basic 'classic' code site focusing primarily on utilizing the Windows APIs from VB. With contents gleaned from poring over the MSDN and other API references, as well as provided by friends and contributors, VB<i>net</i> is nearing the
        80 million page-view mark.</dd>
        <dd class="url">vbnet.mvps.org</dd>

        <dt><a href="http://ccrp.mvps.org/">Common Controls Replacement Project</a></dt>
        <dd>The <em><a href="http://ccrp.mvps.org/">CCRP</a></em> <strike><font color="#808080">is</font></strike> was a unique project, combining the extensive talents of Windows developers whose aim <strike><font color="#808080">is</font></strike> was to provide smaller, faster, more fully-featured and free replacements to the Microsoft-provided Common Controls and Common Dialogs. The untimely end was not foreseen.</dd>
        <dd class="url">ccrp.mvps.org</dd>

        <dt>Eric DeBrosse's <a href="/vbdx/">DirectX Cool Fool</a></dt>
        <dd>A great resource for Visual Basic developers utilizing DirectX technologies. Provides downloads, articles, tutorials and FAQs related to multimedia and game development topics.</dd>

        <dt>Robert Dunlop's <a href="/directx/">X-Zone</a></dt>
        <dd>A useful resource for game developers and others using the DirectX SDK for multimedia development. Includes samples, how-to articles, FAQs for the various DirectX APIs, and more! Also includes support information for Sam's <cite>Teach Yourself DirectX 7 in 24 Hours</cite>, by co-author Robert Dunlop.</dd>

        <dt>L.J. Johnson's <a href="/st-software/">Slightly Tilted Software</a></dt>
        <dd>All of the code from articles that L.J. has done for <a href="http://www.vbpj.com" target="_blank"><cite>VBPJ</cite></a> over the years. Mainly NT administrative-type programs and utilities, but many are useful for anyone who uses NT and Visual Basic. Extensive use of some of the more esoteric API programming under NT (including file security). A sampling of WMI and ADSI programming is also included, along with photos of other MVPs and well-known VB personalities from VBits.</dd>

        <dt>Felix Kasza's <a href="http://win32.mvps.org/">Win32 Samples</a></dt>
        <dd>The <em><a href="http://win32.mvps.org/">Win32</a></em> page contains a variety of Win32 samples, covering areas such as LSA, MIDL and RPC, Network, NTFS streams, Processes, Security, and a few others. Note that Felix does not speak VB.</dd>
        <dd class="url">win32.mvps.org</dd>

        <dt>David Liske's <a href="http://frogleg.mvps.org/helptechnologies/">HTML Help Center</a></dt>
        <dd>This subweb contains the results of research conducted in the area of Microsoft HTML Help. Developers and help authors alike will find high-quality samples, source code, and tools for working with HTML Help, along with a couple of items to assist the users of help systems.</dd>
        <dd class="url">frogleg.mvps.org</dd>

        <dt>Brad Martinez' <a href="http://btmtz.mvps.org/">VB-32 Programs &amp; Samples</a></dt>
        <dd>A unique collection of 32-bit freeware programs and code samples catering to the intermediate and advanced Visual Basic programmer familiar with the Win32 shell common controls, and API.</dd>
        <dd class="url">btmtz.mvps.org</dd>

        <dt>Bob Moore's <a href="http://bobmoore.mvps.org/">Win32 Tip Site</a></dt>
        <dd>An eclectic selection of Win32 programming tips in C/C++. Topics covered range from UI techniques through to processes and threads, with each tip trying to solve a common problem or explain a concept. Basic MFC knowledge useful for some of the tips, but not essential.</dd>
        <dd class="url">bobmoore.mvps.org</dd>

        <dt>Eduardo Morcillo's <a href="/emorcillo/">Edanmo's VB Page</a></dt>
        <dd>Source code for Visual Basic 6 and Visual Basic .NET, targeted to intermediate and advanced programmers.</dd>

        <dt>Karl E. Peterson's <a href="http://vb.mvps.org/">One-Stop Source Shop</a></dt>
        <dd>Dedicated to Classic VB programming, this subweb contains a collection of the samples, demos, and tools that Karl has put together over the years. Many were originally written for <a href="http://www.ftponline.com/archives/magazines/vsm/">VBPJ</a> and have since been enhanced. Others are simply FAQs. If you use Classic VB (this is a certifiably .NET-free zone) you're likely to find something useful here.</dd>
        <dd class="url">vb.mvps.org</dd>

        <dt>Bryan Stafford's <a href="/vbvision">vbVision</a></dt>
        <dd>A collection of Visual Basic samples, articles and tips with what may be considered a slightly different vision than other VB sites. The site is dedicated to exposing the limitless possibilities that are achievable using VB and the Win32 API.</dd>
