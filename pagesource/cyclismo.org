The R language is a free implementation of the S programming language. It can be found at <a href="http://www.r-project.org/">http://www.r-project.org/</a>.</p>

		<h2>MATLAB</h2>
		<p>MATLAB is an environment associated with scientific computing and numerical linear algebra. It is an environment that offers its own programming language, memory management, and a number of libraries to extend the features available. MATLAB is a commerical software product, and more information can be found at <a href="http://www.mathworks.com/">http://www.mathworks.com/</a>. A free implementation of the MATLAB language can be found at <a href="https://www.gnu.org/software/octave/">https://www.gnu.org/software/octave/</a>.</p>
