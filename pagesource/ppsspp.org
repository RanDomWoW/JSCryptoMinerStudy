          <h2>News</h2>
          <a name="news"></a>
          <h3>February 6, 2018: Back in the Play Store!</h3>
          <p>The appeal against the suspension was successful, as expected. We're back!</p>
          <h3>Febraury 1, 2018: News update: PPSSPP temporarily gone from Play Store, OpenGL threading success</h3>
          <p>PPSSPP (not PPSSPP Gold, for some reason) was suspended by Google from the Play Store for unclear reasons. I am appealing the decision. In the meantime, you can download the APK and install it manually.</p>
          <p>In other news, the new OpenGL threading code is turning out to be very fast, nearly reaching Vulkan speeds in some cases. So if games are running slow on your device, and it has more than one core - stay tuned for 1.6! It will still be a while though before the code is fully stable, but it's already showing lots of promise.</p>
          <h3>December 5, 2017: PPSSPP 1.5.4, - more crashfixes, mostly Vulkan</h3>
          <p>Another quick update to take care of some common crashes.</p>
          <p><a href="downloads.html"><img alt="PPSSPP icon" src="img/ppsspp-icon.png" width="24" height="24"> Download PPSSPP 1.5.4!</a></p>
					<p><img alt="PPSSPP Gold icon" src="img/ppsspp-icon-gold.png" width="24" height="24"><a href="https://central.ppsspp.org/buygold"> Buy PPSSPP Gold 1.5.4!</a>
          <h3>November 30, 2017: PPSSPP 1.5.2, - fixes for common crashes</h3>
          <p>Released a quick update to take care of some common crashes in 1.5.</p>
          <h3>November 29, 2017: PPSSPP 1.5 - Vulkan is here!</h3>
          <p>1.5 has support for Vulkan, the new faster way to draw graphics. Supported on many modern Android devices, and if enabled (change Backend in Graphics settings) you will see a massive speed boost. PPSSPP 1.5 also has the usual assortment of fixes:</p>
          <ul>
            <li>Full Vulkan support, also for Android now. Very fast on supported devices. (#10033, #10049)</li>
            <li>Smarter graphics state management, reduced CPU consumption on all backends (#9899)</li>
            <li>Android: Support for Arabic and other scripts we couldn't support before</li>
            <li>Fix Android widgets, screen scaling (#10145)</li>
            <li>Fixes to video dumping</li>
            <li>Geometry problems fixed in Medal of Honor</li>
            <li>Implement immediate draws, fixing Thrillville (#7459)</li>
            <li>Software rendering improvements, speed and accuracy</li>
            <li>Hardware tesselation of PSP Beziers and Splines (used by a few games)</li>
            <li>Partial sceUsbGps and sceUsbCam support (Android)</li>
            <li>Android "Sustained performance mode" to avoid thermal throttling (#9901)</li>
            <li>Linux controller mapping fixes (#9997)</li>
            <li>Assorted bugfixes and compatibility improvements</li>
          </ul>
          <p>Note that Google Play might take a day or two to update, I'm doing a slow rollout this time to catch unexpected crashes from the new Vulkan backend etc. Some issues have been found already, fixes will be made.<p>
