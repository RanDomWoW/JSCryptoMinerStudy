<p style="text-align: left; ">SoftEther VPN Project&nbsp;develops and distributes&nbsp;<strong><a title="Why SoftEther VPN?" rel="internal" href="https://www.softether.org/1-features">SoftEther VPN</a></strong>,<br /> <em>An Open-Source Free &#8203;Cross-platform Multi-protocol VPN Program</em>,<br /> as an academic project from&nbsp;<a class="external" title="http://www.tsukuba.ac.jp/english/" rel="external nofollow" href="http://www.tsukuba.ac.jp/english/" target="_blank">University of Tsukuba</a>.</p>
<div id="section_1"><span id="What_is_SoftEther_VPN"></span><h2 class="editable">What is SoftEther VPN</h2>
<p><a title="1.2.jpg" rel="internal" href="https://www.softether.org/@api/deki/files/4/=1.2.jpg"><img align="right" alt="1.2.jpg" class="internal rwrap" style="width: 350px; height: 322px;" src="https://www.softether.org/@api/deki/files/4/=1.2.jpg?size=webview" /></a><a title="Why SoftEther VPN?" rel="internal" href="https://www.softether.org/1-features"><strong>SoftEther VPN</strong></a> <em>(&quot;SoftEther&quot; means &quot;Software Ethernet&quot;)</em> is one of the world's most powerful and easy-to-use multi-protocol VPN software. It runs on Windows, Linux, Mac, FreeBSD and Solaris.</p>
<p>SoftEther VPN is <a title="Source Code" rel="internal" href="https://www.softether.org/5-download/src">open source</a>. You can use SoftEther for any personal or commercial use for free charge.</p>
<p>SoftEther VPN is an optimum <a title="http://www.softether.org/1-features/1._Ultimate_Powerful_VPN_Connectivity#Support_OpenVPN_Protocol" rel="internal" href="https://www.softether.org/1-features/1._Ultimate_Powerful_VPN_Connectivity#Support_OpenVPN_Protocol">alternative to OpenVPN </a>and <a title="http://www.softether.org/1-features/1._Ultimate_Powerful_VPN_Connectivity#Support_Microsoft_SSTP_VPN_Protocol" rel="internal" href="https://www.softether.org/1-features/1._Ultimate_Powerful_VPN_Connectivity#Support_Microsoft_SSTP_VPN_Protocol">Microsoft's VPN servers</a>. SoftEther VPN has a clone-function of OpenVPN Server. You can integrate from OpenVPN to SoftEther VPN smoothly. <a title="1.3.jpg" rel="internal" href="https://www.softether.org/@api/deki/files/12/=1.3.jpg">SoftEther VPN is faster than OpenVPN</a>. SoftEther VPN also supports Microsoft SSTP VPN for Windows Vista / 7 / 8.&nbsp;No more need to pay expensive&nbsp;charges for&nbsp;Windows Server license for Remote-Access VPN function.</p>
<p>SoftEther VPN can be used to realize <a class="external" title="http://en.wikipedia.org/wiki/Bring_your_own_device" rel="external nofollow" href="http://en.wikipedia.org/wiki/Bring_your_own_device" target="_blank">BYOD</a> (Bring your own device) on your&nbsp;business.&nbsp;If you have smartphones, tablets or laptop PCs, <a title="L2TP/IPsec Setup Guide for SoftEther VPN Server" rel="internal" href="https://www.softether.org/4-docs/2-howto/9.L2TPIPsec_Setup_Guide_for_SoftEther_VPN_Server">SoftEther VPN's L2TP/IPsec server function</a>&nbsp;will help you to establish a remote-access VPN from your local network. SoftEther VPN's L2TP VPN Server has strong compatible with <a title="Windows L2TP Client Setup" rel="internal" href="https://www.softether.org/4-docs/2-howto/9.L2TPIPsec_Setup_Guide_for_SoftEther_VPN_Server/4.Windows_L2TP_Client_Setup">Windows</a>, <a title="Mac OS X L2TP Client Setup" rel="internal" href="https://www.softether.org/4-docs/2-howto/9.L2TPIPsec_Setup_Guide_for_SoftEther_VPN_Server/5.Mac_OS_X_L2TP_Client_Setup">Mac</a>, <a title="iPhone / iPad L2TP Client Setup" rel="internal" href="https://www.softether.org/4-docs/2-howto/9.L2TPIPsec_Setup_Guide_for_SoftEther_VPN_Server/2.iPhone_iPad_L2TP_Client_Setting">iOS</a> and <a title="Android L2TP Client Setup" rel="internal" href="https://www.softether.org/4-docs/2-howto/9.L2TPIPsec_Setup_Guide_for_SoftEther_VPN_Server/3.Android_L2TP_Client_Setup">Android</a>.</p>
<p><a title="1.0_vpnserver.jpg" rel="internal" href="https://www.softether.org/@api/deki/files/8/=1.0_vpnserver.jpg"><img align="left" alt="1.0_vpnserver.jpg" class="internal lwrap" style="width: 350px; height: 263px;" src="https://www.softether.org/@api/deki/files/8/=1.0_vpnserver.jpg?size=webview" /></a>SoftEther VPN is not only an alternative VPN server to existing VPN products (OpenVPN, IPsec and MS-SSTP). SoftEther VPN has also <a title="1. Ultimate Powerful VPN Connectivity" rel="internal" href="https://www.softether.org/1-features/1._Ultimate_Powerful_VPN_Connectivity">original strong SSL-VPN protocol</a> to penetrate any kinds of firewalls. Ultra-optimized SSL-VPN Protocol of SoftEther VPN has very fast throughput, low latency and firewall resistance.</p>
<p>SoftEther VPN has<a title="1. Ultimate Powerful VPN Connectivity" rel="internal" href="https://www.softether.org/1-features/1._Ultimate_Powerful_VPN_Connectivity"> strong resistance against firewalls</a> than ever. <a title="Why SoftEther VPN?" rel="internal" href="https://www.softether.org/1-features/1._Ultimate_Powerful_VPN_Connectivity#1.5._NAT_Traversal">Built-in NAT-traversal</a> penetrates your network admin's troublesome firewall for overprotection. You can setup your own VPN server behind the firewall or NAT in your company, and you can reach to that VPN server in the corporate private network from your home or mobile place, without any modification of firewall settings. Any deep-packet inspection firewalls cannot detect SoftEther VPN's transport packets as a VPN tunnel, because SoftEther VPN uses Ethernet over HTTPS for camouflage.</p>
<p><a title="1.0_vpnclient2.jpg" rel="internal" href="https://www.softether.org/@api/deki/files/7/=1.0_vpnclient2.jpg"><img align="right" alt="1.0_vpnclient2.jpg" class="internal rwrap" style="width: 239px; height: 119px;" src="https://www.softether.org/@api/deki/files/7/=1.0_vpnclient2.jpg?size=webview" /></a>Easy to imagine, design and implement your VPN topology with SoftEther VPN. It virtualizes <a title="2. Layer-2 Ethernet-based VPN" rel="internal" href="https://www.softether.org/1-features/2._Layer-2_Ethernet-based_VPN">Ethernet by software-enumeration</a>. SoftEther VPN virtualizes Ethernet devices in order to realize a flexible virtual private network for both <a title="Remote Access to LAN" rel="internal" href="https://www.softether.org/4-docs/2-howto/1.VPN_for_On-premise/2.Remote_Access_VPN_to_LAN">remote-access VPN</a> and <a title="LAN to LAN Bridge" rel="internal" href="https://www.softether.org/4-docs/2-howto/1.VPN_for_On-premise/3.LAN_to_LAN_Bridge_VPN">site-to-site VPN</a>. SoftEther VPN implements the Virtual Network Adapter program as a software-emulated traditional Ethernet network adapter. SoftEther VPN implements the Virtual Ethernet Switch program (called <a title="3.4 Virtual Hub Functions" rel="internal" href="https://www.softether.org/4-docs/1-manual/3._SoftEther_VPN_Server_Manual/3.4_Virtual_Hub_Functions">Virtual Hub</a>) as a software-emulated traditional Ethernet switch. SoftEther VPN implements VPN Session as a software-emulated Ethernet cable between the network adapter and the switch.</span></p> <p>You can create one or many <a title="3.4 Virtual Hub Functions" rel="internal" href="https://www.softether.org/4-docs/1-manual/3._SoftEther_VPN_Server_Manual/3.4_Virtual_Hub_Functions">Virtual Hub</a> with SoftEther VPN on your server computer. This server computer will become a <a title="3. SoftEther VPN Server Manual" rel="internal" href="https://www.softether.org/4-docs/1-manual/3._SoftEther_VPN_Server_Manual">VPN server</a>, which accepts VPN connection requests from <a title="4. SoftEther VPN Client Manual" rel="internal" href="https://www.softether.org/4-docs/1-manual/4._SoftEther_VPN_Client_Manual">VPN client</a> computers.</p> <p>You can create one or many <a title="4.3 Virtual Network Adapter" rel="internal" href="https://www.softether.org/4-docs/1-manual/4._SoftEther_VPN_Client_Manual/4.3_Virtual_Network_Adapter">Virtual Network Adapter</a> with SoftEther VPN on your client computer. This client computer will become a VPN client, which establishes a VPN connections to the Virtual Hub on the VPN server.</p> <p>You can establish VPN sessions, as called 'VPN tunnels', between VPN clients and VPN servers. A VPN session is the virtualized network cable. A VPN session is realized over a TCP/IP connection. The signals through the VPN session is encrypted by SSL. Therefore, you can safely establish a VPN session beyond the Internet. A VPN session is established by SoftEther VPN's <a title="1. Ultimate Powerful VPN Connectivity" rel="internal" href="https://www.softether.org/1-features/1._Ultimate_Powerful_VPN_Connectivity">&quot;VPN over HTTPS&quot; technology</a>. It means that SoftEther VPN can create a VPN connection beyond <a title="Penetrates Firewall by SSL-VPN" rel="internal" href="https://www.softether.org/4-docs/2-howto/7.Replacements_of_Legacy_VPNs/1.Penetrates_Firewall_by_SSL-VPN">any kinds of firewalls and NATs</a>.</p> <p><a title="1.0.2.jpg" rel="internal" href="https://www.softether.org/@api/deki/files/683/=1.0.2.jpg"><img align="right" alt="1.0.2.jpg" class="internal rwrap" style="width: 350px; height: 216px;" src="https://www.softether.org/@api/deki/files/683/=1.0.2.jpg?size=webview" /></a></p> <p>The Virtual Hub exchanges all Ethernet packets from each connected VPN session to other connected sessions. The behavior is same to traditional Ethernet switches. The Virtual Hub has a FDB (forwarding database) to optimize the transmission of Ethernet frames.</p> <p>You can define a <a title="3.6 Local Bridges" rel="internal" href="https://www.softether.org/4-docs/1-manual/3._SoftEther_VPN_Server_Manual/3.6_Local_Bridges">local bridge</a> between the Virtual Hub and the existing physical Ethernet segment by using the Local Bridge function. The Local Bridge exchanges packets between the physical Ethernet adapter and the Virtual Hub. You can realize a <a title="Remote Access to LAN" rel="internal" href="https://www.softether.org/4-docs/2-howto/1.VPN_for_On-premise/2.Remote_Access_VPN_to_LAN">remote-access VPN</a> from home or mobile to the company network by using the Local Bridge function.</p> <p>You can define a <a title="10.5 Build a LAN-to-LAN VPN (Using L2 Bridge)" rel="internal" href="https://www.softether.org/4-docs/1-manual/A._Examples_of_Building_VPN_Networks/10.5_Build_a_LAN-to-LAN_VPN_(Using_L2_Bridge)">cascading connection</a> between two or more remote Virtual Hubs. With cascading, you can integrate two or more remote Ethernet segments to a single Ethernet segment. For example, after you establish cascading connections between the site A, B and C, then any computers in the site A will be able to communicate with the computers in the site B and the site C. This is a <a title="LAN to LAN Bridge" rel="internal" href="https://www.softether.org/4-docs/2-howto/1.VPN_for_On-premise/3.LAN_to_LAN_Bridge_VPN">site-to-site VPN</a>.</p> <p>SoftEther VPN can also establish a VPN session over UDP. The UDP-mode of SoftEther VPN supports <a title="Dynamic DNS and NAT Traversal" rel="internal" href="https://www.softether.org/4-docs/2-howto/6.VPN_Server_Behind_NAT_or_Firewall/1.Dynamic_DNS_and_NAT_Traversal">NAT traversal</a>. The NAT traversal function allows the VPN server behind existing NATs or firewalls to accept incoming VPN sessions. You need no network administrator's special permission before setting up a VPN server on the company network behind firewalls or NATs. Additionally, SoftEther VPN Server may be placed on the dynamic IP address environment since SoftEther VPN has built-in <a title="Dynamic DNS and NAT Traversal" rel="internal" href="https://www.softether.org/4-docs/2-howto/6.VPN_Server_Behind_NAT_or_Firewall/1.Dynamic_DNS_and_NAT_Traversal">Dynamic DNS (DDNS)</a> function.</p> <p>SoftEther VPN Server supports additional VPN protocols, including <a title="L2TP/IPsec Setup Guide for SoftEther VPN Server" rel="internal" href="https://www.softether.org/4-docs/2-howto/9.L2TPIPsec_Setup_Guide_for_SoftEther_VPN_Server">L2TP/IPsec</a>, <a title="Replacements of OpenVPN" rel="internal" href="https://www.softether.org/4-docs/2-howto/7.Replacements_of_Legacy_VPNs/2.Replacements_of_OpenVPN">OpenVPN</a>, <a title="1. Ultimate Powerful VPN Connectivity" rel="internal" href="https://www.softether.org/1-features/1._Ultimate_Powerful_VPN_Connectivity">Microsoft SSTP</a>, <a title="1-features/1._Ultimate_Powerful_VPN_Connectivity" rel="internal" href="https://www.softether.org/1-features/1._Ultimate_Powerful_VPN_Connectivity">L2TPv3</a> and <a title="1-features/1._Ultimate_Powerful_VPN_Connectivity" rel="internal" href="https://www.softether.org/1-features/1._Ultimate_Powerful_VPN_Connectivity">EtherIP</a>. These realizes the interoperability with <a title="L2TP/IPsec Setup Guide for SoftEther VPN Server" rel="internal" href="https://www.softether.org/4-docs/2-howto/9.L2TPIPsec_Setup_Guide_for_SoftEther_VPN_Server">built-in L2TP/IPsec VPN clients on iPhone, iPad, Android, Windows and Mac OS X</a>, and also with <a title="Cisco L2TPv3/IPsec Edge-VPN Router Setup" rel="internal" href="https://www.softether.org/4-docs/2-howto/9.L2TPIPsec_Setup_Guide_for_SoftEther_VPN_Server/6.Cisco_IOS_L2TPv3%2F%2F%2F%2FIPsec_Edge-VPN_Router_Setup">Cisco's VPN routers</a> and other vendors VPN products.</p>
